## 五、eraAssetsService.ts 素材服务

`services/assets/eraAssetsService.ts` 提供了：
- `loadEraManifest(eraId)` — 加载 manifest.json
- `checkEraAssetsStatus(eraId)` — 检查就绪状态（ready/pending/missing/unknown）
- `getEraBgm(eraId)` — 获取BGM路径
- `loadEraAssets(eraId)` — 加载完整素材

**CDN URL 构造规则**：
```
${R2_CDN_BASE}/data/era_assets/${eraId}/manifest.json
${R2_CDN_BASE}/data/era_assets/${eraId}/${bgmFile}
${R2_CDN_BASE}/data/era_assets/${eraId}/${imageFile}
```

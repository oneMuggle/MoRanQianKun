# 用户功能清单

> 更新日期：2026-05-03
> 本文档从用户角度描述已完成功能，易于非技术人员理解。

## 已完成功能

### 1. 剧情生成与交互
- **功能描述**：玩家输入动作，AI生成剧情回应，支持回合式交互和动作选项选择
- **使用方式**：在底部输入框输入文字，点击发送或按Enter；AI提供动作选项时点击选择
- **相关文档**：`docs/user-manual/user-manual.md`

### 2. 开局创建流程
- **功能描述**：新游戏向导，依次选择时代、体质、门派、境界、天赋、背景，AI生成开局剧情
- **使用方式**：点击「新游戏」→ 选择时代 → 选择体质/门派/境界 → 选择天赋/背景 → 确认开局
- **相关文档**：`docs/user-manual/user-manual.md`

### 3. NSFW内容控制
- **功能描述**：四级场景分级（无/点到为止/适度展开/完全展开）+ 三级内容过滤，安全可控的成人内容系统
- **使用方式**：在新游戏向导中选择nsfw场景类型，设置中可开启成人内容开关
- **相关文档**：`docs/nsfw-system-design.md`

### 4. 亲密度社交系统
- **功能描述**：与NPC的亲密度等级（0-5级），随好感度提升解锁不同亲密动作（调情→拥抱→抚摸→双修）
- **使用方式**：通过送礼、对话、切磋等互动提升好感度，解锁更高亲密等级
- **相关文档**：`docs/nsfw-system-design.md`

### 5. 角色属性与成长
- **功能描述**：境界突破、功法修炼、装备穿戴、物品使用、天赋效果生效
- **使用方式**：点击顶部栏角色名称查看属性面板，通过战斗和修炼提升境界
- **相关文档**：`docs/talent-background-system.md`

### 6. 世界演变系统
- **功能描述**：游戏世界根据时间和玩家行为动态演变，NPC状态、势力、事件自动推进
- **使用方式**：点击「世界演变」按钮手动触发，或等待游戏内时间自动推进
- **相关文档**：`docs/世界观系统分析.md`

### 7. 世界书编辑
- **功能描述**：编辑世界设定，添加自定义规则，注入NPC信息，修改场景描述
- **使用方式**：点击「世界书」按钮 → 编辑 → 保存；支持按作用域（开局/主剧情/世界演变等）分别注入
- **相关文档**：`docs/世界观系统分析.md`

### 8. 图像生成系统
- **功能描述**：为角色（主角/NPC）、场景生成图像，支持NovelAI/SD/WebUI等多个后端
- **使用方式**：打开社交/角色面板 → 找到「生成图像」/「生图」按钮 → 等待AI生成
- **相关文档**：`docs/user-manual/user-manual.md`

### 9. 背包与物品管理
- **功能描述**：存储和使用物品（装备武器、防具、饰品；使用药品等消耗品）
- **使用方式**：点击右侧栏「背包」按钮 → 选中物品 → 点击「使用」或「穿戴」
- **相关文档**：`docs/user-manual/user-manual.md`

### 10. 门派系统
- **功能描述**：创建门派、升级门派、招收弟子、管理门派事务
- **使用方式**：点击右侧栏「门派」按钮 → 选择创建/升级/招收等操作
- **相关文档**：`docs/user-manual/user-manual.md`

### 11. 任务系统
- **功能描述**：主线任务和支线任务的查看与进度追踪
- **使用方式**：点击右侧栏「任务」按钮 → 查看任务列表和进度
- **相关文档**：`docs/user-manual/user-manual.md`

### 12. 地图系统
- **功能描述**：查看当前位置和世界地理分布，切换到其他已解锁地点
- **使用方式**：点击右侧栏「地图」按钮 → 选择目的地
- **相关文档**：`docs/user-manual/user-manual.md`

### 13. 记忆系统
- **功能描述**：查看游戏记忆摘要，手动触发记忆召回，回顾重要剧情节点
- **使用方式**：点击右侧栏「记忆」按钮 → 查看摘要或触发召回
- **相关文档**：`docs/user-manual/user-manual.md`

### 14. 存档与云同步
- **功能描述**：本地IndexedDB存档 + GitHub云端同步（可选，需配置GitHub OAuth）
- **使用方式**：「保存进度」/「读取进度」按钮本地操作；「云存档」按钮配置GitHub后自动同步
- **相关文档**：`docs/云同步实现与配置分析.md`、`docs/user-manual/user-manual.md`

### 15. 战斗系统
- **功能描述**：回合制战斗，选择攻击/防守/功法/物品等动作，属性判定战斗结果
- **使用方式**：点击右侧栏「战斗」按钮进入战斗模式，选择动作
- **相关文档**：`docs/user-manual/user-manual.md`

### 16. 约定系统
- **功能描述**：自定义游戏规则，设置生成偏好，添加特殊约束
- **使用方式**：点击右侧栏「约定」按钮 → 设置规则和偏好
- **相关文档**：`docs/user-manual/user-manual.md`

### 17. 队伍系统
- **功能描述**：创建队伍、招收队员、分配队伍任务
- **使用方式**：点击右侧栏「队伍」按钮 → 管理队伍
- **相关文档**：`docs/user-manual/user-manual.md`

### 18. 社交系统
- **功能描述**：与NPC互动、查看NPC资料、生图、赠送礼物、提升好感度
- **使用方式**：点击右侧栏「社交」按钮 → 选择NPC → 互动/生图/送礼
- **相关文档**：`docs/user-manual/user-manual.md`

### 19. 女主规划系统
- **功能描述**：女主剧情规划、镜头排期、互动排期
- **使用方式**：点击右侧栏「女主规划」按钮 → 查看/管理女主剧情进度
- **相关文档**：`docs/剧情规划-女主规划-小说分解-动态世界联动重构计划.md`

### 20. 酒馆预设系统
- **功能描述**：导入和管理AI角色预设，配置提示词注入策略（未选择/单一用户/严格/半严格）
- **使用方式**：设置中心 → 酒馆预设 → 导入/启用/配置
- **相关文档**：`docs/technical/prompt-system.md`

### 21. 委婉词过滤
- **功能描述**：自动将明确描写替换为委婉表达（玉茎→花径，春宵一度→巫山云雨等）
- **使用方式**：系统自动生效，根据NSFW档位自动切换
- **相关文档**：`docs/nsfw-system-design.md`

### 22. 搜索过滤功能
- **功能描述**：在新游戏向导中快速搜索和过滤天赋、背景、气运
- **使用方式**：新游戏向导 → 在搜索框输入关键词 → 过滤结果显示
- **相关文档**：`docs/plans/search-filter-new-game-wizard.md`

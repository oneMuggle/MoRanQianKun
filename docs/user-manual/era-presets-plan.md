# 时代适配天赋/背景/气运 计划

## 需求

由于现在有多个时代的设定，开局设置中的天赋、背景、气运等都需要增加适配时代的条目，后续将根据时代筛选这些条目。

## 现状盘点

| 数据集 | 总量 | 古代专属 | 通用 | 近代专属 | 现代专属 | 近未来专属 | 未来专属 |
|---|---|---|---|---|---|---|---|
| **天赋** | 55 | ~41 | 13 | 0 | 0 | 0 | 0 |
| **背景** | 60 | ~56 | 3 | 0 | 0 | 0 | 0 |
| **气运** | 250 | ~219 | 25 | 0 | 0 | 0 | 0 |

**通用条目（全时代可见）**：
- 天赋(13)：过目不忘、福星高照、平平无奇、稳扎稳打、人情练达、耐苦心性、账房脑子、静心观微、守拙藏锋、权谋心眼、伪装皮相、赌命胆魄、麒麟臂
- 背景(3)：山野孤儿、寒门子弟、商贾之家
- 气运(25)：现代梗/概念类（拼夕夕砍一刀、Ctrl+Z法则、直播修仙、弹幕护体等）

> 注意：时代适配使用 **时代背景大类**（古代/近代/现代/近未来/未来）而非具体 SubEra ID，这样同一背景大类下的 SubEra 共享内容池。

## 目标

每个时代的**专属条目**数量大致均衡，加上通用条目后总可选数量在同一量级。

| 数据集 | 古代(现有) | 近代(目标) | 现代(目标) | 近未来(目标) | 未来(目标) |
|---|---|---|---|---|---|
| 天赋 | 41+13通用 | 15+13通用 | 15+13通用 | 15+13通用 | 15+13通用 |
| 背景 | 56+3通用 | 12+3通用 | 12+3通用 | 12+3通用 | 12+3通用 |
| 气运 | 219+25通用 | 50+25通用 | 50+25通用 | 50+25通用 | 50+25通用 |

> 古代气运数量远超其他时代（修仙网文梗天然丰富），短期不削减，后续逐步补齐。

## 实施步骤

### Phase 0：基础设施（本期可交付）

#### Step 0.1：扩展类型定义

在以下接口中新增 `时代适配?: string[]`：

- `types.ts` — `天赋结构`、`背景结构`
- `data/qiyun/index.ts` — `气运数据`

#### Step 0.2：为现有条目打时代标签

**天赋打标**：
- 41 条古代专属：加 `时代适配: ['古代']`
- 13 条通用：不打标
- 1 条 NSFW 通用（如适用）：不打标

**背景打标**：
- 56 条古代专属：加 `时代适配: ['古代']`
- 3 条通用：不打标

**气运打标**：
- 219 条古代专属（真气运/限制版/合欢秘辛等修仙/武侠主题）：加 `时代适配: ['古代']`
- 25 条现代梗/概念类（直播修仙、拼夕夕、Ctrl+Z 等）：不打标
- 6 条边界条目（需判断）：暂时不打标，保持可用

#### Step 0.3：wizard 过滤逻辑

修改 `useNewGameWizardState.ts` 中的三个 useMemo，在现有性别+NSFW过滤基础上增加：

```typescript
const 当前时代 = worldConfig.时代配置ID;
const 时代背景 = 获取时代背景(当前时代); // '古代' | '近代' | '现代' | '近未来' | '未来'
const 匹配时代 = (item: { 时代适配?: string[] }) =>
    !item.时代适配 || item.时代适配.length === 0 || item.时代适配.includes(时代背景);
```

需要新增 `获取时代背景()` 工具函数：将 SubEra ID 映射到时代背景大类。

#### Step 0.4：UI 可选提示

在天赋/背景/气运列表项上，对有限制的条目显示小标签（如"古代专属"），帮助用户理解筛选结果。

## 风险评估

- **Phase 0 零风险**：可选字段，不打标的条目全时代可见
- **Phase 1-4 工作量**：每个 Phase 约 77 条创意编写+数据录入，可分批交付
- **气运数量不对等**：古代气运 219 条远超目标 50，短期不影响体验（玩家选 3 条），长期可考虑为其他时代创作更多

## 后续优化

- 按 SubEra 细化：在时代背景基础上增加 SubEra 级别的 `时代适配`（如某个天赋仅适用于 `ancient_eastern_wuxia`）
- 气运均衡：古代气运数量远超其他时代，可为其他时代追加更多主题气运
- 动态推荐：根据选择的 SubEra，高亮推荐最相关的天赋/背景/气运

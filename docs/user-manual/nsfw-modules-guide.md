# NSFW 模块用户指南

> 本文档汇总墨色江湖中所有 NSFW（成人向）功能模块的使用说明。

## 模块概览

本项目包含多个 NSFW 子系统，按时代背景和功能主题分类：

### 校园纪元 NSFW

- **校园欲望系统** — NPC 欲望档案、欲望等级、互动事件
- **校园论坛深夜板块** — BDSM 子论坛、寻主召奴招募系统
- **校园祭 NSFW** — 特殊活动期间的限定内容

### 都市纪元 NSFW

- **成人产业** — 演艺圈、直播/短视频、写真约拍
- **高端会所/SPA** — 按摩服务、VIP 体验
- **夜场/KTV 生态** — 夜场互动、KTV 玩法
- **婚恋相亲** — 约会、恋爱关系发展
- **整形/医美** — 外貌改变、整形事件
- **潜水/水上运动** — 水上活动场景
- **野外/极限运动** — 户外冒险场景
- **宠物经济** — 宠物互动事件
- **糖爹/糖宝关系** — 特殊人际关系模式

### BDSM 系统

- **BDSM 关系/互动管线** — 角色关系、互动场景、状态管理
- **多角色 BDSM 关系网** — 复杂关系网络
- **桌游社交 NSFW** — 桌游场景中的成人内容
- **写真系统** — 写真拍摄、模特档案、摄影师系统

### 使用方式

各 NSFW 模块通过以下方式访问：

1. **游戏配置** — 在设置中启用/禁用 NSFW 模式
2. **NPC 互动** — 通过社交面板与 NPC 互动触发
3. **论坛系统** — 通过手机论坛应用访问
4. **桌游面板** — 通过桌游社交界面
5. **写真面板** — 通过写真系统入口

### 内容分级

项目支持多级 NSFW 内容控制，详见 `nsfw-systems-guide.md`。

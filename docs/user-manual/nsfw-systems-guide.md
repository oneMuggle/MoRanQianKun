# NSFW 系统使用指南：BDSM & 露出

> 日期: 2026-05-16
> 适用版本: 校园+现代+古代全时代

## 概述

BDSM NSFW 和露出 NSFW 现已成为独立的 NSFW 子系统，可在所有时代（校园、现代、古代等）独立启用和配置。两个系统通过 NSFW 中心统一管理，各有独立的设置面板和仪表盘。

## 启用方式

1. 打开游戏主界面
2. 进入 **NSFW 中心**
3. 找到 **BDSM NSFW** 或 **露出 NSFW** 模块卡片
4. 点击主开关启用
5. 点击设置图标配置子选项

## BDSM NSFW 系统

### 设置项（12 个开关）

| 分类 | 开关 | 说明 |
|------|------|------|
| **主开关** | 启用 BDSM 独立系统 | 总开关，关闭时所有 BDSM 内容不生效 |
| **基础** | BDSM 内容强度 | 关闭/轻度(心理暗示)/中度(明确行为)/深度(详细场景) |
| **论坛** | 启用 BDSM 论坛 | 开启后包含匿名讨论、寻主召奴等子分类 |
| **关系管线** | 启用 BDSM 关系管线 | 开启后追踪关系状态、服从度、权力天平等 |
| **关系管线** | 启用 BDSM 调教任务 | 开启后 AI 每回合生成调教任务/日常指令 |
| **关系管线** | 启用 BDSM 契约系统 | 开启后可与 NPC 协商并缔结 BDSM 契约 |
| **关系管线** | 启用 BDSM 见面预约 | 开启后可与 NPC 预约见面时间与地点 |
| **高级** | 启用 BDSM 多角色关系 | 支持多 NPC 之间的 BDSM 关系网络 |
| **高级** | 启用 BDSM 时代场景包 | 根据当前时代自动匹配场景和修正值 |
| **高级** | 启用 BDSM 信誉系统 | 论坛中积累信誉影响寻主/召奴效果 |
| **高级** | 启用 BDSM 安全词历史 | 记录每次使用安全词的场景和原因 |
| **高级** | 启用 BDSM 契约模板库 | 提供预设契约模板供快速缔结 |

### 依赖关系

- 调教任务、契约系统、见面预约依赖「关系管线」开启
- 其余功能独立于关系管线，只要主开关开启即可使用

## 露出 NSFW 系统

### 设置项（6 个开关）

| 开关 | 说明 |
|------|------|
| **启用露出系统** | 总开关，开启后引入紧张度、旁观者、网络传播 |
| **露出内容强度** | 关闭/轻度(私密半公开)/中度(半公共场所)/深度(公共场所) |
| **启用公开隐秘侵犯** | 开启后在公共场合可能触发秘密互动 |
| **启用旁观者反应** | 开启后周围人可能注意到并触发流言/后果 |
| **启用网络传播** | 开启后可能有偷拍、论坛讨论、社交媒体传播 |
| **活动 NSFW 频率** | 关闭/低(仅特殊活动)/中(中型活动)/高(所有活动) |

### 依赖关系

- 「旁观者反应」依赖「公开隐秘侵犯」开启
- 「网络传播」依赖「旁观者反应」开启
- 其余功能独立

### 仪表盘

启用露出系统后，可在 NSFW 中心的露出仪表盘查看：

- **NPC 露出档案**：每个 NPC 的露出等级、进度条、成功/失败次数、紧张度值、网络流言等级
- **旁观者记录**：周围人类型、距离、是否已察觉、反应描述

## 系统间联动

BDSM NSFW 和露出 NSFW 之间可以联动。例如：
- 露出系统中的旁观者可能触发 BDSM 相关的网络流言
- BDSM 系统的论坛帖子可能讨论露出相关内容

两个系统的联动是自动的，无需额外配置。

## 旧存档兼容

如果你之前使用过校园 NSFW 系统中的露出功能，旧存档会自动迁移：

- 旁观者类型 "普通同学" → "路人"，"闺蜜好友" → "密友" 等
- 旁观者距离 "同桌邻座" → "紧邻"，"对面附近" → "附近" 等
- 周围人状态 "正常上课" → "专注事务"

迁移是自动且不可逆的，建议在迁移前备份存档。

## 与校园 NSFW 的关系

校园 NSFW 系统仍然保留以下校园专属子系统：
- 基础设置（NSFW 内容强度、后果系统、多角关系）
- SM/支配服从系统（校园版）
- 桌游社交 NSFW（密室逃脱、狼人杀、剧本杀、派对游戏）
- 校园祭 NSFW（后夜祭告白、摊位 NSFW、舞台 NSFW）

露出和 BDSM 的独立系统可以在校园场景中使用，也可以在现代、古代等其他场景中使用。校园 NSFW 中不再包含露出和 BDSM 的设置开关，但它们仍通过模块消费关系与校园 NSFW 联动。

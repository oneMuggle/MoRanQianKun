## 1. 背景与动机

当前项目中，「古代武侠」时代拥有完整的子体系选择机制：
- **武侠**：传统武学体系
- **志怪**：超自然妖怪/异闻体系
- **双修**：武侠 + 志怪共存

每个体系还有对应的「里模式」开关（里武侠、里志怪），开启后注入 NSFW 向的深层规则提示词。

但其他时代（现代都市、近未来赛博、未来科幻等）没有类似的子体系选择机制。本方案为「现代都市」时代添加同等级别的三选一子体系 + 里模式系统。

## 2. 现有体系架构回顾

### 2.1 古代体系的数据流

```
用户选择 时代(古代武侠)
  → NewGameWizard 展示 体系选择按钮(武侠/志怪/双修)
    → 用户选择 体系(如:志怪)
      → 可选开启 里模式(里武侠/里志怪)
        → 保存时写入 IndexedDB:
          - 古代体系选择: '志怪'
          - 启用里志怪模式: true/false
          - 启用里武侠模式: true/false

运行时系统提示词注入 (systemPromptBuilder.ts):
  - 里武侠开启 → 注入 构建里武侠世界提示词()
  - 里志怪开启 → 注入 构建里志怪世界提示词()
  - 体系=志怪 且 里志怪未开 → 注入 构建志怪世界提示词()
  - 数据裁剪链: 修炼体系 → 里武侠 → 里志怪

世界生成注入 (worldSetup.ts):
  - isAncient && 体系=志怪 → 注入 构建志怪世界提示词()
  - 里模式 → 用 构建<Feature>附加块() 包裹
```

### 2.3 里武侠提示词结构参考

里武侠 (`prompts/runtime/liWuxiaWorld.ts`) 的核心内容：

1. **表里双修原理**：公开武学 vs 阴阳修行
2. **武根系统**：硬度、尺寸、精元储量三属性
3. **角色表里人格**：外表端庄/内在放浪的反差
4. **衣着描写规则**：暴露、走光、破损
5. **行为模式规则**：主动挑逗、性吸引力
6. **功法表里描述格式**：每个功法须含表里两面

新的现代里模式需要遵循这个结构模式，但内容完全现代化。

## 3. 现代体系设计

### 3.1 三个子体系概览

| 体系 | 主题 | 核心玩法 | 世界观基调 |
|------|------|----------|------------|
| **普通都市** | 现代都市写实 | 职场博弈、社交网络、资本运作、信息战 | 现实向的都市生活与权力博弈 |
| **末日避难** | 末日灾变求生 | 资源管理、避难所建设、生存技能、人性考验 | 灾变后的世界重建与人性挣扎 |
| **丧尸生存** | 丧尸末世求生 | 丧尸对抗、安全区防守、物资搜刮、感染防治 | 恐怖末世的生存与变异 |

### 3.2 体系切换的影响

切换体系后影响以下层面：

1. **UI 层**：新游戏向导展示不同的体系选择按钮和世界观预设卡片
2. **提示词层**：注入不同的世界观规则提示词
3. **世界生成层**：世界生成时使用不同的规则约束
4. **数据裁剪层**：体系特有数据字段在关闭时被裁剪

### 3.3 与古代体系的对应关系

| 古代 | 现代对应 |
|------|----------|
| 武侠 | 普通都市（基础生存/竞争体系） |
| 志怪 | 丧尸生存（超自然/变异体系） |
| 双修 | 末日避难（混合体系，兼有现实与极端） |

## 4. 里模式 NSFW 设计

每个现代体系都有独立的里模式开关，开启后注入 NSFW 向的深层规则。

### 4.1 里普通都市模式

**触发条件**：`现代体系选择 === '普通都市'` + `启用里普通都市模式 === true`

**核心规则**：

1. **身份反差原理**
   - 表身份：白天是白领/高管/教师/医生等正经职业
   - 里身份：夜晚的欲望消费者/夜场玩家/暗网参与者
   - 反差越大，角色越有张力

2. **权色交易体系**
   - 职场中的权力与性交换
   - 资本运作中的身体资本化
   - 信息即权力，身体也是信息载体

3. **欲望消费主义**
   - 夜场文化、私人会所、地下派对
   - 消费主义延伸到身体与情感
   - "一切皆可商品化"的世界观

4. **暗网潜规则**
   - 隐藏的社交网络和交易渠道
   - 匿名身份下的真实欲望释放
   - 数字时代的新型亲密关系

5. **衣着与行为规则**
   - 职场正装的隐秘性感（紧衬衫、包臀裙、丝袜、高跟鞋）
   - 脱下正装后的反差（夜店装、私密装）
   - 商务场合中的暗语和暗示

6. **角色表里人格**
   - 白天：专业、理性、克制
   - 夜晚：主动、放浪、追求刺激
   - 职场中的权力博弈延伸到亲密关系

### 4.2 里末日避难模式

**触发条件**：`现代体系选择 === '末日避难'` + `启用里末日避难模式 === true`

**核心规则**：

1. **废土狂化原理**
   - 文明崩塌后的人性释放
   - 法律与道德约束失效
   - 生存本能驱动一切行为

2. **身体作为交易筹码**
   - 资源稀缺 → 身体成为硬通货
   - 避难所内的权力与性分配
   - 以亲密关系换取食物、水、安全

3. **避难所权力与性**
   - 避难所首领的特权
   - 派系之间的联姻/交换
   - 底层居民的无奈与妥协

4. **极端环境下的人性**
   - 堕落与希望并存
   - 在绝望中寻找温暖
   - 亲密关系成为生存的精神支柱

5. **衣着描写规则**
   - 破损、拼凑的末日服装
   - 功能性衣物中的意外走光
   - 以物易物换来的"奢侈品"

6. **角色表里人格**
   - 表：坚强、独立、为生存而战
   - 里：脆弱、渴望被保护、用身体换取安全感

### 4.3 里丧尸生存模式

**触发条件**：`现代体系选择 === '丧尸生存'` + `启用里丧尸生存模式 === true`

**核心规则**：

1. **感染变异原理**
   - 丧尸病毒的多种变异形态
   - 感染者的身体异变与欲望扭曲
   - 半感染者的挣扎与异化

2. **安全区内的权力与性交易**
   - 安全区守卫的特权
   - 以身体换取进入安全区的资格
   - 幸存者团体的内部关系

3. **生存压力下的亲密关系**
   - 末世中的短暂温存
   - "明日可能死去"的紧迫感
   - 在死亡阴影下的情感爆发

4. **变异者的原始冲动**
   - 半变异者的意识残留
   - 变异带来的感官变化
   - 人性与兽性的边界

5. **衣着描写规则**
   - 战斗/逃生导致的破损
   - 防护服下的身体曲线
   - 搜刮来的不匹配衣物

6. **角色表里人格**
   - 表：冷静、警惕、为生存而战
   - 里：渴望温暖、用亲密确认自己还活着

7. **体系专属概念** — 该体系独有的系统/机制

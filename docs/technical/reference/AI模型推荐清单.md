# 项目独立 API 功能 AI 模型推荐清单

## 概述

本文档为项目中需要独立 AI API 的各个规划功能提供模型推荐，以供用户参考。

基于代码分析，项目共有 **19 个 AI 调用函数**，分布在：
- 文本生成：`services/ai/text/storyTasks.ts` (13 个)
- 图片生成：`services/ai/image/imageTasks.ts` (5 个)
- 接口测试：`testConnection` (1 个)

## 一、功能详细模型推荐

### 1. 主剧情生成 (generateStoryResponse)

| 项目 | 详情 |
|------|------|
| **功能** | 游戏核心循环，每次用户输入后生成剧情正文 |
| **输入特征** | 用户输入 + 历史记录(数十轮/数千token) + 系统提示词 + 酒馆预设 + 上一个响应 |
| **输出要求** | 带 `<thinking>`、`<正文>`、`<命令>`、`<行动选项>` 等结构化标签；要求强推理能力、角色一致性、剧情连贯性 |
| **推荐模型** | **Claude 3.5 Sonnet** (Primary) / GPT-4o (备选) |
| **廉价替代** | GPT-4o-mini (仅用于低优先级备用) |
| **推荐理由** | 核心体验，强推理、长上下文、JSON/结构化输出能力要求高 |
| **配置优先级** | ⭐⭐⭐⭐⭐ (最高) |

### 2. 世界观生成 (generateWorldData)

| 项目 | 详情 |
|------|------|
| **功能** | 生成游戏世界观母本，包含地理、势力、社会秩序等 |
| **输入特征** | 世界设定上下文 + 角色数据 + 配置参数 |
| **输出要求** | 输出 `<世界观>` 标签块，要求 JSON 严格格式遵循，自洽可长期使用 |
| **推荐模型** | **Claude 3.5 Sonnet** 或 **Gemini 2.5 Pro** |
| **廉价替代** | GPT-4o |
| **推荐理由** | 需要严格格式、结构化输出、长程一致性 |
| **配置优先级** | ⭐⭐⭐⭐ (高) |

### 2.2 API 独立配置建议

项目支持在设置页配置独立 API，建议按以下方式分配：

#### 方案 A：单 API 方案 (入门)
- 使用 **同一个主力模型** (如 Claude 3.5 Sonnet) 处理所有功能
- 优点：简单、成本可控
- 缺点：无法针对特定功能优化

#### 方案 B：双 API 方案 (推荐)
- **主 API**：Claude 3.5 Sonnet / GPT-4o
  - 主剧情生成、世界演变、规划分析、世界观生成、变量校准、小说分解
- **图片 API**：NovelAI / Midjourney
  - 图片生成相关功能

#### 方案 C：三 API 方案 (进阶)
- **主剧情 API**：Claude 3.5 Sonnet
  - 主剧情生成、世界演变、规划分析
- **世界观 API**：Gemini 2.5 Pro
  - 世界观生成、变量校准、小说分解
- **图片 API**：NovelAI
  - 图片生成

#### 方案 D：四 API 方案 (完备)
- **主剧情 API**：Claude 3.5 Sonnet
- **世界观API**：Claude 3.5 Sonnet / Gemini 2.5 Pro
- **辅助 API**：GPT-4o-mini
  - 记忆召回、正文润色、提示词生成
- **图片 API**：NovelAI

## 三、具体模型推荐 (2025)

### 3.1 主力模型 (Primary)

| 模型 | 供应商 | 优势 | 适用场景 |
|------|--------|------|----------|
| **Claude 3.5 Sonnet** | Anthropic | 超长上下文(200K)、推理强、JSON输出好 | 主剧情、世界演变、规划分析、小说分解 |
| **Gemini 2.5 Pro** | Google | 超长上下文(1M)、性价比高、多模态 | 世界观生成、长文本处理 |
| **GPT-4o** | OpenAI | 平衡发展、工具调用强 | 变量校准、正文润色、图片提示词 |

### 3.2 廉价模型 (Budget)

| 模型 | 供应商 | 优势 | 适用场景 |
|------|--------|------|----------|
| **GPT-4o-mini** | OpenAI | 速度快、成本低、能力强 | 接口测试、简单润色、记忆召回 |
| **Claude 3.5 Haiku** | Anthropic | 速度快、成本低 | 简单检索、基础理解 |

### 3.3 图片生成模型

| 模型 | 供应商 | 优势 | 适用场景 |
|------|--------|------|----------|
| **NovelAI** | 专用 | 二次元/动漫风格顶级 | 角色立绘、NPC、场景 |
| **Midjourney** | 专用 | 写实/艺术风格强 | 场景、氛围图 |
| **DALLE-3** | OpenAI | 生态集成好 | 通用图像 |
| **Stable Diffusion** | 开源 | 可自托管、成本低 | 本地部署 |

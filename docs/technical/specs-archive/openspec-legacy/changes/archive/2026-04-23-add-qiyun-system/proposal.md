## Why

references/气运.md 包含 300+ 个精心设计的气运设定，涵盖真·气运系列、限制版气运、因果律系列等多个类别，是极具特色的武侠/修仙创意素材。将这些气运与当前项目结合，可为角色创建、开局生成、NPC 设定提供丰富的天赋系统选择，大幅增强游戏的可玩性和随机性。

## What Changes

1. **新增气运数据模块**：将气运设定结构化存储，项目可调用
2. **角色气运选择功能**：开局/创角时可选择气运作为初始天赋
3. **气运效果系统**：气运对角色属性、战斗、事件产生实际影响
4. **NPC 气运生成**：NPC 随机分配气运，增加世界丰富度
5. **排除不适内容**：不包含「女性向」类别中的成人内容

## Capabilities

### New Capabilities
- `qiyun-data`: 气运数据结构化存储，按类别/稀有度索引
- `qiyun-selection`: 角色开局选择气运作为天赋系统
- `qiyun-effects`: 气运效果的实际计算与生效
- `qiyun-npc-generation`: NPC 创建时随机分配气运

### Modified Capabilities
- `character-creation`: 增加气运选择阶段
- `world-generation`: 世界生成时考虑 NPC 气运分布
- `stats-system`: 属性计算加入气运修正因子

## Impact

- 新增 `data/qiyun/` 数据目录
- 修改 `models/` 中的角色/NPC 类型定义
- 扩展 `prompts/` 相关的角色创建提示词
- 扩展 `hooks/` 中角色属性计算逻辑
- 排除 references/气运.md 中的「女性向(30个)」章节

## Scope

### 包含范围
- 真·气运 (强力效果) - 约 90 个
- 限制版气运 (挑战模式) - 约 120+ 个
- 沙雕控场·因果律系列 - 约 10 个
- 搞钱薅羊毛·天道规则系列 - 约 10 个
- 苟命隐身·绝对无敌系列 - 约 10 个
- 奇葩攻击·脑洞破防系列 - 约 10 个
- 概念篡改·法则扭曲系列 - 约 10 个
- 偷鸡摸狗·白嫖躺赢系列 - 约 10 个
- 摆烂摸鱼·怠惰降维系列 - 约 6 个
- 奇葩buff·精神暴击系列 - 约 25 个

### 非目标
- 不实现「女性向」类别（内容不适配项目定位）
- 不实现完整的气运技能系统（仅实现基础效果）
- 不包含气运获取/更换 gameplay（初始一次性选择）
- 不做平衡性微调（玩家自行选择，风险自担）

## Acceptance Criteria

1. 气运数据按类别、稀有度可检索
2. 角色创建时可从气运列表中选择 1-3 个初始气运
3. 气运对角色属性有可量化的修正因子
4. NPC 生成时随机分配气运
5. UI 显示角色的气运条目
6. 不包含「女性向」内容

## ADDED Requirements

### Requirement: 角色气运选择界面
系统 SHALL 在角色创建流程中提供气运选择功能。

#### Scenario: 查看可选气运列表
- **WHEN** 玩家进入气运选择阶段
- **THEN** 显示气运列表，每页N个，支持分类筛选

#### Scenario: 选择初始气运
- **WHEN** 玩家点击选择某个气运
- **THEN** 该气运被添加到玩家气运列表，选择数量限制1-3个

#### Scenario: 确认气运选择
- **WHEN** 玩家点击「确认」按钮且已选择至少1个气运
- **THEN** 气运选择结果写入角色数据，进入下一阶段

### Requirement: 气运选择限制
系统 SHALL 限制气运选择数量。

#### Scenario: 最少选择1个气运
- **WHEN** 玩家尝试跳过气运选择
- **THEN** 提示「请至少选择1个气运」

#### Scenario: 最多选择3个气运
- **WHEN** 玩家已选择3个气运后尝试再选择
- **THEN** 提示「最多选择3个气运」，阻止添加

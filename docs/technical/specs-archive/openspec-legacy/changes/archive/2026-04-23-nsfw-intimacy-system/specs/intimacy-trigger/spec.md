# Intimacy Trigger Specification

## ADDED Requirements

### Requirement: Location-based trigger
The system SHALL offer intimacy options when player visits specific locations.

#### Scenario: Player enters trigger location
- **WHEN** player enters a location with intimacy trigger
- **THEN** system SHALL display interaction options appropriate to NPC present

### Requirement: Event-based trigger
The system SHALL trigger intimacy options based on game events.

#### Scenario: Special event occurs
- **WHEN** a story event, festival, or NPC state change occurs
- **THEN** system MAY display special intimacy options if conditions met

### Requirement: Player-initiated trigger
The system SHALL allow players to actively initiate intimacy interactions.

#### Scenario: Player clicks initiate button
- **WHEN** player clicks "亲密互动" button on NPC panel
- **THEN** system SHALL display available intimacy options based on NPC's intimacy level

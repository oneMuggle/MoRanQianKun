# NSFW 亲密互动系统提案

## Why

墨染乾坤目前的互动主要停留在叙事层面，缺少与角色深度互动的机制。对比参考文件《大渊习武录》和《姬侠传 V0.4.0》，需要建立完整的亲密互动系统，让玩家与 NPC 的关系有更深层次的发展，同时将亲密互动与角色成长（功法/属性）结合，提升游戏沉浸感。

## What Changes

### 新增能力
- **亲密度系统**：0-100 好感度分级，解锁不同互动等级
- **触发式互动**：基于地点/事件/NPC 状态的条件触发
- **表里双修**：表层正常互动 + 里层亲密互动（亲密度解锁）
- **亲密奖励**：互动触发属性提升/功法领悟
- **NSFW 模式开关**：可切换的互动范围模式
- **互动UI面板**：触摸/亲密动作选择卡片

### 变更范围
- 扩展 useGame 状态管理，添加亲密度数据结构
- 新增亲密互动提示词模板
- 扩展 NPC 面板显示亲密度状态
- 新增设置开关控制互动范围

### 非目标
- **不做**具体 NSFW 内容生成（由 AI 模型决定）
- **不做**复杂的肢体物理模拟
- **不做**强制触发机制（始终基于自愿）

## Capabilities

### New Capabilities
- `intimacy-system`: 亲密度分级与状态管理（0-100 分级）
- `intimacy-trigger`: 基于场景/NPC/事件的触发系统
- `dual-layer-interaction`: 表层互动（正常）+ 里层互动（亲密解锁）
- `intimacy-reward`: 亲密互动奖励（属性/功法/内力）
- `intimacy-ui`: 亲密互动选择面板
- `content-mode-switch`: SFW/NSFW 模式开关

### Modified Capabilities
- `social-npc`: 扩展现有 NPC 结构，增加亲密度字段

## Impact

### 受影响模块
- `hooks/useGame/` - 需扩展亲密度状态
- `models/` - 新增亲密度类型定义
- `components/features/Social/` - 扩展 NPC 面板
- `prompts/` - 新增亲密互动提示词
- `utils/settings` - 新增模式开关

### 验收标准
1. 亲密度显示 0-100 数值，与 NPC 面板集成
2. 亲密度每 20 点解锁一级互动（共5级）
3. 3 种触发条件：地点/事件/主动
4. 亲密互动可触发属性奖励
5. 设置中可切换 SFW/NSFW 模式
6. 互动选择 UI 面板正常显示

## Decisions

### 1. 能力类型分类规则

根据效果描述判断：

| 能力类型 | 判断规则 | 示例 |
|---------|---------|------|
| 战斗 | 直接提升攻击/防御/爆发 | 真·荒古圣体、真·吞噬进化 |
| 生存 | 保命/恢复/防御/复活 | 真·复活甲、真·绝对闪避 |
| 社交 | 人脉/声望/威吓/谈判 | - |
| 谋略 | 情报/判断/预知/布局 | 真·人生剧本、真·源天师 |
| 特殊 | 奇遇/运气/随机/特殊机制 | 真·每日签到、真·退婚流 |
| 辅助 | 修炼/效率/资源/成长 | 真·挂机修仙、真·氪金成仙 |

### 2. 适用境界分类规则

按气运效果强度分级：

| 境界层级 | 境界名 | 适用气运类型 |
|---------|--------|-------------|
| 0 | 凡人 | 基础保命/废材逆袭类 |
| 1-9 | 炼气期 | 普通气运 |
| 10-19 | 筑基期 | 中等强度气运 |
| 20-29 | 金丹期 | 高强度气运 |
| 30+ | 元婴及以上 | 顶级气运/因果律 |

**规则**：
- 描述效果为「新手起步」「废材逆袭」→ [0, 9]
- 普通战斗/修炼气运 → [1, 29]
- 高强度气运（如吞噬进化、神象镇狱）→ [10, 99]
- 奇遇类气运（退婚流、九龙拉棺）→ [0, 99]
- 因果律类气运 → [20, 99]

### 3. NSFW 标记规则

```typescript
nsfw等级: 0 | 1 | 2
// 0 = 无 NSFW 内容（默认）
// 1 = 轻度（性暗示、软暴力）
// 2 = 重度（明确性描写、极端暴力、伦理禁忌）
```

**标记标准**：
- 涉及性行为、性暗示 → nsfw等级 ≥ 1
- 极度暴力、血腥描述 → nsfw等级 = 2
- 伦理禁忌内容 → nsfw等级 = 2

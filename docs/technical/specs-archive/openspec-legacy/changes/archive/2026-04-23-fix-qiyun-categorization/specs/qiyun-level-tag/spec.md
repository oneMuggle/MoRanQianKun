## ADDED Requirements

### Requirement: 气运适用境界范围标注
每个气运 SHALL 标注其适用的境界层级范围。

#### Scenario: 标注格式验证
- **WHEN** 气运已标注 `适用境界`
- **THEN** `适用境界` 为 `[number, number]` 格式
- **AND** `适用境界[0]` ≤ `适用境界[1]`

#### Scenario: 凡人境界标注
- **WHEN** 气运适用于凡人（境界层级0）
- **THEN** `适用境界[0]` ≤ 0

#### Scenario: 全境界标注
- **WHEN** 气运适用于所有境界
- **THEN** `适用境界[0]` = 0 且 `适用境界[1]` ≥ 99

### Requirement: 境界层级映射
系统 SHALL 遵循标准的境界层级映射。

#### Scenario: 境界层级映射正确性
- **WHEN** 需要判断气运适用境界
- **THEN** 使用以下映射表：
  | 境界 | 层级范围 |
  |------|---------|
  | 凡人 | 0 |
  | 炼气初期 | 1-3 |
  | 炼气中期 | 4-6 |
  | 炼气后期 | 7-9 |
  | 筑基初期 | 10-12 |
  | 筑基中期 | 13-15 |
  | 筑基后期 | 16-19 |
  | 金丹期 | 20-29 |
  | 元婴及以上 | 30+ |

### Requirement: 境界适配与气运强度匹配
气运的适用境界 SHALL 与其强度匹配。

#### Scenario: 高强度气运境界限制
- **WHEN** 气运效果为「因果律」「绝对无敌」「逆天改命」
- **THEN** `适用境界[0]` ≥ 20

#### Scenario: 中等强度气运境界
- **WHEN** 气运效果为普通战斗/修炼加成
- **THEN** `适用境界[0]` 在 1-10 之间

#### Scenario: 入门级气运境界
- **WHEN** 气运效果为「废材逆袭」「新手起步」
- **THEN** `适用境界[1]` ≤ 9

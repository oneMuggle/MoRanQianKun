## Why

当前 NSFW 标记为二元（是/否），但「女性向」内容中包含程度差异较大的条目。部分为一般向可接受，部分为重口向（涉及明显性描写、轮奸、调教等）需要额外设置才能开放。

## What Changes

1. **NSFW 分级**：将 NSFW 分为「一般向」和「重口向」两级
2. **重口标记**：明确标记需要额外设置的内容
3. **解锁机制**：重口内容需主动开启「成人内容」设置

## Capabilities

### New Capabilities
- `nsfw-level`: NSFW 内容分级（0=安全，1=一般，2=重口）
- `nsfw-unlock`: 重口内容解锁设置

### Modified Capabilities
- `qiyun-filter`: 增加分级过滤参数
- `天赋/背景过滤`: 增加分级过滤

## Scope

### 分级标准
| 等级 | 描述 | 示例 |
|------|------|------|
| 0=安全 | 无 NSFW 内容 | 福星高照 |
| 1=一般向 | 轻微暧昧，性暗示 | 风月玲珑 |
| 2=重口向 | 明显性描写，轮奸，调教 | 「女性向」中部分条目 |

### 重口向特征（以下任一）：
- 明确性行为描写
- 轮奸/强暴情节
- 调教/物化描写
- 极端身体改造
- 失去人格的描写

### 非目标
- 不修改现有安全内容
- 不删除任何已有数据

## Acceptance Criteria

1. NSFW 内容分为三级（0/1/2）
2. 一般向（1）可直接显示
3. 重口向（2）需设置开启
4. 过滤函数支持分级参数
5. 原有 nsfw 标记兼容

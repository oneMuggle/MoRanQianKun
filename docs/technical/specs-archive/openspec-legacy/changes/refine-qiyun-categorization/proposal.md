## Why

气运数据需要与项目的能力系统和武力等级适配。目前气运设定约300个未分类，直接使用会导致生成提示词过长、效果难以量化。此外，原References中部分重口味内容需要筛选排除。

## What Changes

1. **能力类型分类**：按气运效果类型分类（战斗/生存/社交/谋略/特殊/辅助）
2. **武力等级适配**：气运设定标注适用的境界层级范围（凡人/炼气/筑基/金丹等）
3. **NSFW内容过滤**：明确标识并排除不适合项目调性的内容
4. **效果量化**：为可量化气运标注属性修正值

## Capabilities

### New Capabilities
- `qiyun-category`: 气运按能力类型分类
- `qiyun-level-adaptation`: 气运标注适用境界范围
- `qiyun-filter`: NSFW/重口内容标记与过滤

### Modified Capabilities
- `qiyun-data`: 增加类别、能力类型、境界适配字段
- `qiyun-effects`: 效果标注为可量化/描述性分类

## Scope

### 分类维度
能力类型：
- 战斗类（直接提升战力的气运）
- 生存类（保命/恢复/防御相关）
- 社交类（人脉/声望/威吓相关）
- 谋略类（情报/判断/布局相关）
- 特殊类（奇遇/运气/稀有机制）
- 辅助类（修炼/效率/资源相关）

境界适配：
- 凡人（境界层级0）
- 炼气（境界层级1-9）
- 筑基（境界层级10-19）
- 金丹（境界层级20-29）
- 元婴及以上（境界层级30+）

### 非目标
- 不降低气运数量上限（可筛选非删除）
- 不修改气运原有效果设定
- 不做平衡性调整

## Acceptance Criteria

1. 每个气运标注所属类别和能力类型
2. 每个气运标注适用境界范围
3. 明确区分可量化效果与描述效果
4. NSFW内容可按标记过滤
5. 分类后数据可按维度检索

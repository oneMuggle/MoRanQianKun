## ADDED Requirements

### Requirement: 古代预设数据
古代背景使用现有的预设天赋和背景数据。

#### Scenario: 古代天赋选择
- **WHEN** 时代为「古代」显示天赋列表时
- **THEN** 显示现有 presets.ts 中的天赋选项

#### Scenario: 古代背景选择
- **WHEN** 时代为「古代」显示背景列表时
- **THEN** 显示现有预设背景(门派、江湖、官府等)

### Requirement: 现代预设数据
现代背景使用新时代的预设数据。

#### Scenario: 现代天赋选择
- **WHEN** 时代为「现代」显示天赋列表时
- **THEN** 显示现代背景的天赋(如：金融直觉、黑客天赋、格斗本能等)

#### Scenario: 现代背景选择
- **WHEN** 时代为「现代」显示背景列表时
- **THEN** 显示现代背景(公司高层、黑帮成员、警务人员等)

### Requirement: 未来预设数据
未来背景使用科幻时代的预设数据。

#### Scenario: 未来天赋选择
- **WHEN** 时代为「未来」显示天赋列表时
- **THEN** 显示未来背景的天赋(如：义体适配、黑客直觉、星际航行等)

#### Scenario: 未来背景选择
- **WHEN** 时代为「未来」显示背景列表时
- **THEN** 显示未来背景(赛博浪客、星际船员、人工智能工程师等)

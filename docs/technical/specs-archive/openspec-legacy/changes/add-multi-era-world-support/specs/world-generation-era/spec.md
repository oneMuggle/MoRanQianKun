## ADDED Requirements

### Requirement: 时代感知世界生成
世界生成系统根据选择的时代动态加载对应提示词。

#### Scenario: 古代世界生成
- **WHEN** 时代设置为「古代」触发世界生成时
- **THEN** 使用古代逻辑提示词和古代世界模板生成世界观

#### Scenario: 现代世界生成
- **WHEN** 时代设置为「现代」触发世界生成时
- **THEN** 使用现代逻辑提示词和现代世界模板生成世界观

#### Scenario: 未来世界生成
- **WHEN** 时代设置为「未来」触发世界生成时
- **THEN** 使用未来逻辑提示词和未来世界模板生成世界观

### Requirement: 世界生成时的时代检测
世界生成时读取存档中的时代配置。

#### Scenario: 读取存档时代配置
- **WHEN** 加载存档并触发世界演变/重新生成时
- **THEN** 系统读取存档的游戏设置中的时代配置

#### Scenario: 时代配置缺失的回退
- **WHEN** 存档无时代配置时
- **THEN** 默认使用「古代」时代

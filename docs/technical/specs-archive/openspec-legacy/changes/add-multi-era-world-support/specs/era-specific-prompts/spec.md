## ADDED Requirements

### Requirement: 古代时代逻辑提示词
生成古代武侠世界观时使用古代现实逻辑。

#### Scenario: 古代世界生成
- **WHEN** 设置时代为「古代」且生成世界时
- **THEN** 加载古代逻辑提示词(礼制、江湖组织、门派、帮派等)

#### Scenario: 古代背景NPC生成
- **WHEN** 时代为「古代」生成NPC时
- **THEN** NPC使用古代称谓、身份、门派等背景

### Requirement: 现代时代逻辑提示词
生成现代世界观时使用现代社会逻辑。

#### Scenario: 现代世界生成
- **WHEN** 设置时代为「现代」且生成世界时
- **THEN** 加载现代逻辑提示词(法律、公司、金融、枪械等)

#### Scenario: 现代背景NPC生成
- **WHEN** 时代为「现代」生成NPC时
- **THEN** NPC使用现代身份、公司/帮派组织、金融关系等背景

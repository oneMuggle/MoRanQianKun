## ADDED Requirements

### Requirement: 游戏设置可选择时代
用户新建游戏或修改设置时可选择时代背景。

#### Scenario: 新建游戏时选择古代
- **WHEN** 用户点击新建游戏并选择「古代」时代
- **THEN** 创建的存档使用古代时代配置

#### Scenario: 新建游戏时选择现代
- **WHEN** 用户点击新建游戏并选择「现代」时代
- **THEN** 创建的存档使用现代时代配置

#### Scenario: 新建游戏时选择未来
- **WHEN** 用户点击新建游戏并选择「未来」时代
- **THEN** 创建的存档使用未来时代配置

#### Scenario: 修改现有存档时代
- **WHEN** 用户在设置中修改时代选项
- **THEN** 存档的时代配置更新，刷新世界生成

### Requirement: 时代默认值
未设置时代时默认使用古代。

#### Scenario: 旧存档无时代配置
- **WHEN** 打开一个在添加时代选项前创建的存档
- **THEN** 系统默认使用「古代」时代

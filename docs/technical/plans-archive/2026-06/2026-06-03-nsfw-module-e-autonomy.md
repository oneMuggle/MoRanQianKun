## 一、设计目标

将当前"仅注入 AI 提示词"的 NPC 自主行为系统，升级为**玩家可感知、可选择回应、可预判**的互动玩法。

**核心设计原则**：
1. NPC 不是被动对象——他们有欲望、有动机、会主动行动
2. 玩家能看到 NPC 的日程和意图——但不会看到全部（保持神秘感）
3. 回应 NPC 行为有选择——接受/拒绝/推迟各有后果
4. NPC 之间有社交网络——嫉妒/流言/关系会影响群体动态

### 2.2 NPC 自主行为的 8 种行为完整判定逻辑

**执行流程**：依次运行 7 个判定函数 → 收集所有触发行为 → 选优先级最高的一个返回。

| # | 行为类型 | 触发条件 | 概率公式 | 优先级公式 |
|---|---------|---------|---------|-----------|
| 1 | **主动邀约** | 好感度>=40, 心情>=50 | (好感/100)×(心情/100)×0.4 | 好感×0.6 + 心情×0.4 |
| 2 | **试探行为** | 好感度在 30-60 之间 | 0.2 + (好感-30)/300 | 好感×0.5 |
| 3 | **情绪表达** | 有当前情绪 | 开心15% / 低落或愤怒25% | 开心30 / 负面50 |
| 4 | **嫉妒表现** | 嫉妒状态存在且强度>=20 | 100%（必触发） | 嫉妒强度×0.8 |
| 5 | **NSFW暗示** | 亲密度>=3, 好感>=60, 防线<=70 | 0.05+(亲密度-3)×0.05+(好感-60)/400 | 亲密度×15 + 好感×0.2 |
| 6 | **关心问候** | 羁绊值>=30 | 0.1 + 羁绊值/500 | 羁绊值×0.5 |
| 7 | **记忆触发** | 记忆库有强度>50的关联记忆 | 最强记忆强度/500 | 最强记忆强度×0.6 |

**内在动机**（6 欲望维度）：
- 被认可 / 刺激感 / 安全感 / 归属感 / 支配感 / 自由感
- 满足度 0-100，衰减速率 2-5/回合
- 人格系数：傲娇→被认可1.3+支配感1.2，温柔→归属感1.3+安全感1.2，高冷→自由感1.3，元气→刺激感1.3+归属感1.2，社恐→安全感1.3
- 满足度 < 50 的欲望生成短期目标（最多 3 个）

### 2.3 嫉妒系统完整逻辑

**强度计算公式**：
```
嫉妒值 = (好感度×0.5 + 关系强度×0.3) × 人格系数 × (互动强度/100) × 类型权重
```

**人格系数**：
| 人格 | 系数 |
|------|------|
| 病娇/占有 | 2.0 |
| 傲娇/大小姐 | 1.5 |
| 邻家/清纯 | 1.2 |
| 默认 | 1.0 |
| 高冷/飒爽 | 0.6 |
| 慵懒/佛系 | 0.5 |

**互动类型权重**：NSFW=2.0 / 亲密行为=1.5 / 邀约=1.0 / 礼物=0.8 / 对话=0.5

**6 种表现形式**（按强度分段）：
| 强度 | 表现 | 人格修正 |
|------|------|---------|
| >=80 | 暗中报复 | 病娇→自我伤害 |
| >=60 | 加速推进 | 傲娇→主动质问 |
| >=40 | 冷淡回应 | - |
| >=20 | 冷淡回应 | 心机→拉拢结盟 |

**衰减**：每回合 -5 强度，持续回合 = ceil(强度/10)

## 三、核心玩法循环

### 3.1 回合预处理流程（已实现）

```
每回合开始时调用 执行NSFW回合预处理(npc, 在场NPC列表[], 最近互动类型?, 亲密度等级?)

1. 确保初始化：复合情感 / 内在动机 / 节奏状态
2. 衰减欲望满足度：所有 6 个欲望维度按速率衰减
3. 检测情感冲突：4 对冲突组合
4. 推进前戏阶段：如果有最近互动类型
5. 计算自主行为：对每个在场 NPC 运行 7 个判定函数
6. 合并提示词注入：自主行为提示 + 前戏阶段提示 → AI 提示词
```

**当前问题**：第 5 步的结果只注入 AI 提示词，玩家完全看不到。

### 3.2 深化后的玩家可见流程

```
每回合开始
    ↓
回合预处理（同上）
    ↓
NPC 主动行为判定
    ↓
有 NPC 触发了主动行为？
    ├── 否 → 正常 AI 叙事
    └── 是 → 生成行为事件
                ↓
           玩家选择：
           ├── [查看详情] → 打开 NPC 日程面板
           ├── [立即回应] → 弹出回应选择
           └── [忽略] → 继续叙事，NPC 行为只注入 AI
                ↓
           玩家回应：
           ├── 接受 → 执行 NPC 行为，消耗精力，亲密度变化
           ├── 拒绝 → 亲密度下降，NPC 情绪变化
           └── 推迟 → 下回合可再次选择，小幅亲密度提升
```

```
    ├── 试探行为 → 弹窗通知
    ├── NSFW暗示 → 弹窗通知
    ├── 关心问候 → 弹窗通知（低优先级）
    ├── 记忆触发 → 弹窗通知（低优先级）
    └── 嫉妒表现 → 高优先级通知
    ↓
玩家回应
    ↓
更新 NPC 状态（亲密度/好感度/心情/羁绊/嫉妒）
    ↓
注入 AI 提示词（含玩家回应结果）
```

### 5.3 行为冷却与频率控制

| 行为类型 | 冷却 | 说明 |
|---------|------|------|
| 主动邀约 | 每回合最多 1 个 NPC | 多 NPC 同时触发时选优先级最高的 |
| 试探行为 | 每 2 回合 | 避免过于频繁 |
| 情绪表达 | 每回合 | 情绪波动自然发生 |
| 嫉妒表现 | 持续中必触发 | 嫉妒状态持续期间每回合触发 |
| NSFW暗示 | 每 3 回合 | 较稀有 |
| 关心问候 | 每 2 回合 | 羁绊深时更频繁 |
| 记忆触发 | 每 3 回合 | 稀有但有意义 |
| 日常互动 | 每回合 | 日常可见 |

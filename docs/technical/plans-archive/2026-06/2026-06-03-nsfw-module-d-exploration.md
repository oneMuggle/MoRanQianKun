## 一、设计目标

将当前"纯被动黑盒"的性癖/敏感点发现机制，升级为**玩家可主动参与、有策略选择、有进度反馈**的探索玩法。

**核心设计原则**：
1. 探索本身是一种玩法——玩家做选择（试探方式/深度策略/目标选择）
2. 探索有代价——消耗精力、影响关系、触发防线反弹
3. 探索有进度——玩家能看到线索积累、进度条、解锁预览
4. 探索有风险——失败会下降信任/亲密，防线会反弹

## 三、核心玩法循环

### 3.1 探索流程总览

```
玩家触发探索（NPC 社交面板）
    ↓
1. 选择目标 NPC
    ↓
2. 选择模式：试探交互 / 深度探索 / 敏感点探索
    ↓
3. 设置策略：试探类型/深度策略/探索方式
    ↓
4. 消耗资源：精力 -5~-15，可能消耗代币
    ↓
5. 成功率判定：亲密度/信任度/防线/强度
    ↓
成功 → 发现性癖/敏感点 → 信任+2 → 亲密+3 → AI叙事注入
失败 → 获得线索(进度条) → 信任下降 → AI叙事注入
    ↓
6. 状态更新：性癖演化/防线变化/潜能累积/偏好漂移/里程碑
```

### 3.2 试探交互（低投入、低风险、低回报）

**玩家操作**：
1. 选择试探类型：`言语试探`(成功率×0.8) / `肢体试探`(×0.6) / `场景试探`(×0.7)
2. 确认消耗：精力 -5
3. 系统判定成功率

**成功率公式**（已实现）：
```
base = 0.5
      + 亲密度加成 (最大 0.4)
      + 信任度加成 (最大 0.3)
      - 防线减成 (最大 0.3)
      - 强度减成 ((强度 - 1) × 0.08)
× 试探系数
clamp 到 [0.05, 0.95]
```

**探索策略（影响失败后果）**：
- `安全`：失败时信任 -1，亲密 +0
- `激进`：失败时信任 -3，亲密 -1，但有 30% 概率发现额外线索

**成功结果**：
- 发现一个性癖（从解锁树的"可解锁"节点中选出最佳候选）
- 信任 +2，亲密 +3
- AI 叙事注入：发现过程被描述到故事中

**失败结果**：
- 获得线索（进度条 +15%）
- 信任根据策略下降
- AI 叙事注入：NPC 回避/警惕/反感的描述

**UI 反馈**：
```
┌──────────────────────────────────────┐
│  🔍 试探交互 — [NPC姓名]            │
├──────────────────────────────────────┤
│  试探类型：言语试探                   │
│  探索策略：安全 ○ 激进               │
│  精力消耗：-5                         │
│  预估成功率：68%                      │
│  (亲密度 60 + 信任度 50 - 防线 65)   │
│                                      │
│  [开始试探] [取消]                   │
│                                      │
│  ───── 结果 ─────                    │
│  ✅ 成功！发现性癖：【被关注】        │
│  类别：情感依赖 · 强度 3             │
│  信任 +2 │ 亲密 +3                   │
│                                      │
│  可解锁提示：                         │
│  ⬜ 事后依恋 (需前置: 被关注)         │
└──────────────────────────────────────┘
```

### 3.3 深度探索（中投入、中风险、高回报）

**玩家操作**：
1. 选择一个**已解锁性癖**（或其前置节点）
2. 选择策略：`安全` / `激进` / `暂停`
3. 确认消耗：精力 -12

**目标**：
- 解锁解锁树上的**后置节点**（如"权力交换"的前置是"掌控节奏"）
- 提升已解锁性癖的**强度等级**（上限 5）

**成功率公式**（已实现）：
```
base = 0.4
      + 亲密度加成 (最大 0.3)
      + 信任度加成 (最大 0.2)
      - 强度减成 ((强度 - 1) × 0.1)
× 策略系数: 安全 0.7 / 激进1.2 / 暂停0
clamp 到 [0.05, 0.85]
```

**策略差异**：

| 策略 | 成功率 | 成功奖励 | 失败惩罚 | 适用场景 |
|------|--------|---------|---------|---------|
| 安全 | ×0.7 | 信任+1, 亲密+3 | 信任-1, 亲密+0 | 首次探索新节点 |
| 激进 | ×1.2 | 信任-1, 亲密+1 | 信任-3, 亲密-1 | 需要快速解锁时 |
| 暂停 | ×0 | — | — | 中止当前探索 |

**解锁树示例**：
```
【主导控制】分支：
  ① 掌控节奏 [已解锁] — 强度 2 —▶ 强度 3 (需3次成功)
           │
           ▼ 可解锁
  ② 权力交换 [未解锁] — 前置: ① —▶ 需要 1 次成功
           │
           ▼ 可解锁
  ③ 服从渴望 [未解锁] — 前置: ② —▶ 需要 2 次成功
```

### 3.4 敏感点探索（身体探索玩法）

**玩家操作**：
1. 选择身体区域（头颈区/胸胸区/腰腹区/四肢区/背部区/特殊区）
2. 选择探索方式：`轻抚`(×1.0) / `轻咬`(×1.2) / `温热`(×0.9) / `按压`(×0.8) / `指尖划过`(×1.1)
3. 确认消耗：精力 -10

**已发现敏感点的互动反馈**：
```
综合评分 = 方式系数 × (敏感度/3) × 开发加成 × 随机因子(0.7-1.3)

评分 >= 1.5 → 正面反馈，开发程度 +1，亲密度变化 = 评分×1.5
评分 >= 0.8 → 轻度正面，亲密度 +1
评分 <  0.8 → 中性，无变化
```

**组合技巧**（已实现）：
- 当 NPC 的同一身体区域有两个以上的已发现技巧时，玩家可选择组合使用
- 有协同效果：反应强度 5，亲密度 +6
- 无协同效果：反应强度 3，亲密度 +3

**开发程度**（5级）：
```
未开发 → 初步探索 → 渐入佳境 → 深度开发 → 完全开发
加成: 0.8x     1.0x       1.2x       1.4x       1.6x
```

**新技巧发现**：每次敏感点探索有 30% 概率发现新技巧。

## 五、与游戏循环的集成

### 5.1 探索触发点

```
触发方式：
1. 玩家主动在 NPC 社交面板点击"探索"按钮
2. 每回合 NPC 主动行为中有概率触发"试探交互"（配合/回避）
3. AI 叙事自然引导到探索环节

探索冷却：
- 同一 NPC 的试探交互：每回合 1 次
- 同一 NPC 的深度探索：每 2 回合 1 次
- 同一 NPC 的敏感点探索：每回合 1 次（不同部位可同时）
```

### 5.2 回合结算中的自动流程

```
回合结算时调用：

1. 更新潜能累积(npc, 本回合事件, 时间)
   → 事件匹配触发类别 → 对应潜能条目累积+1

2. 计算偏好漂移(npc)
   → 20事件滑动窗口分析 → 如有漂移则提升对应潜能值

3. 更新心理防线(npc, 本回合事件)
   → 如有 NSFW 事件则扣减防线

4. 应用防线恢复(npc, 当前时间)
   → 每回合恢复 0.5 点

5. 检查人格翻转(npc, 本回合事件, 时间)
   → 防线值 < 25 时检查翻转

6. 检查里程碑(npc, 当前统计)
   → 性癖解锁数/敏感点开发数变化时检查
```

### 5.3 AI 叙事注入

```
系统提示词注入（每回合更新）：

【NPC性癖探索状态】
{NPC姓名} 已发现性癖: [列表+强度]
潜在倾向: [线索分析摘要，最多3条]
心理防线: [等级+描述]

【NPC敏感点状态】
{NPC姓名} 已发现敏感点: [区域+部位+开发程度]
已发现技巧: [技巧列表+协同效果]

【NPC人格演化】
```

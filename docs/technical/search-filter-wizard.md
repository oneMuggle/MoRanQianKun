# 开局设置搜索过滤功能

> **Status:** ✅ 已完成 | **实现日期:** 2026-04-28

## 需求

在开局设置的三个选择区域添加搜索/过滤功能：
1. **背景** — 60+ 项，单选
2. **天赋** — 55+ 项，多选（最多 3 个）
3. **气运** — 250 项，多选（最多 3 个）

此前用户必须滚动所有项目，仅支持性别和 NSFW 切换过滤。

## 已完成内容

### 共用模式：搜索 + 过滤栏

每个区域在折叠区域内部、网格上方获得一致的过滤栏：
1. **文本搜索输入框** — 200ms 防抖，按名称 + 描述过滤
2. **分类/稀有度芯片** — 药丸式切换按钮

### 背景（步骤 2）

- 仅文本搜索（名称 + 描述匹配）
- 60 个项目，搜索即可管理

### 天赋（步骤 2）

- 仅文本搜索（名称 + 描述匹配）
- 55 个项目，与背景类似

### 气运（步骤 2）

- 文本搜索（名称 + 描述匹配）
- **分类过滤芯片** — 11 个分类来自 `气运数据类别列表`: 真·气运, 限制版气运, 因果律, 天道规则, 绝对无敌, 脑洞破防, 法则扭曲, 白嫖躺赢, 怠惰降维, 精神暴击, 合欢秘辛
- **稀有度过滤芯片** — 传说, 稀有, 普通

## 实现细节

### 状态管理

`useNewGameWizardState.ts` 新增 5 个状态变量和 3 个 `useMemo` 计算数组，分别对应背景、天赋、气运的过滤逻辑。

### FilterBar 共享组件

`components/ui/FilterBar.tsx` 包含 `SearchInput`（200ms 防抖）和 `ChipGroup`（药丸式切换按钮）。

### UI 集成

三个区域均接入过滤栏，替换原始数组为过滤后数组，添加"无匹配结果"空状态。

## 变更文件

## 风险评估

- 250 项气运使用 `useMemo` 避免性能问题
- 折叠区域高度需适配过滤栏
- 搜索防抖 200ms 响应感良好

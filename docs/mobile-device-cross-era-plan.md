> 项目：墨色江湖：无尽武林
> 版本：v1.0
> 日期：2026-04-30
> 状态：设计方案，待实施

---

## 一、需求概述

### 1.1 用户需求
1. **现代（都市）**：增加手机功能，支持地图、群聊、论坛等应用
2. **其他时代**：增加类似功能，但形态需符合各时代科技/魔法背景
3. **近未来**：明确定位为"数据终端"

### 1.2 设计原则
- **时代一致性**：设备形态、交互方式需与时代科技水平匹配
- **功能统一性**：核心功能模块在各时代保持一致性，方便用户理解
- **沉浸感优先**：设备名称、界面风格符合时代世界观
- **信息互通**：跨时代信息传递可作为剧情元素

### 2.2 设备详细规格

#### 2.2.1 远古时代 — 石板刻绘 / 骨片记事

```
设备名称：石板刻绘
外观描述：刻有神秘符文的石板，或打磨光滑的骨片
材质：石灰石/玉石/兽骨
尺寸：手掌大小，约15-20cm
通讯范围：部落据点内（50-100米）
能源：无（体力驱动）
储存：约10-20幅简单图案
特性：
  - 可用尖锐物体刻绘
  - 可多层叠加显示
  - 带有神秘力量的符文可发光
```

#### 2.2.2 古代东方 — 传音玉简

```
设备名称：传音玉简 / 千里传音符
外观描述：温润玉石或金属镜面，边缘有符文
材质：玉石/青铜/玄铁
尺寸：巴掌大小，约10-15cm
通讯范围：同城内（武侠）/ 跨城（志怪）
能源：真气/灵力
储存：约100条语音/文字信息
特性：
  - 注入真气激活
  - 支持语音和文字
  - 部分可投影小型地图
  - 可录制场景（需特殊材质）
```

#### 2.2.3 古代西方 — 羊皮信卷 / 神谕镜

```
设备名称：信使卷轴 / 预言铜镜
外观描述：卷起的羊皮纸或带符文的铜镜
材质：莎草纸/羊皮/青铜
通讯范围：城邦之间（信使）/ 神谕范围（有限）
特性：
  - 可加密书写
  - 重要信息用封印蜡封
  - 镜子可显示远方景象（神话属性）
```

#### 2.2.4 近代 — 电报机 / 无线电

```
设备名称：电报收发器 / 无线电台
外观描述：木质外壳的机械装置，带旋钮和指示灯
材质：木材/黄铜/玻璃
尺寸：桌面大小，约30-50cm
通讯范围：城市间（有线）/ 国内（无线电）
能源：电池/交流电
储存：约500条报文
特性：
  - 摩斯电码编解码
  - 可接收广播新闻
  - 部分车载型号
```

#### 2.2.5 近代西方 — 维多利亚时期

```
设备名称：通讯怀表 / 机械信使
外观描述：精美的黄铜机械装置，带小屏幕
材质：黄铜/银/玻璃
通讯范围：数百米（直接）/ 城市范围（中继站）
特性：
  - 机械齿轮驱动
  - 可打印小纸条
  - 带有精美雕刻装饰
```

#### 2.2.6 现代都市 — 智能手机【核心设计】

```
设备名称：智能手机
外观描述：全面屏触屏设备
材质：玻璃/铝合金/塑料
尺寸：约15cm x 7cm
通讯：5G/4G/WiFi/蓝牙
储存：云端+本地
核心应用：
  1. 【地图】— 实时定位、导航、地点标记
  2. 【通讯录】— 联系人管理、关系标记
  3. 【群聊】— 即时通讯、群组功能
  4. 【论坛】— 帖子发布、回复、点赞
  5. 【资讯】— 新闻推送、天气、信息流
  6. 【相册】— 拍照、截图、滤镜
  7. 【工具】— 闹钟、计算器、备忘录
```

#### 2.2.7 当代废土 — 对讲机 / 简易通讯器

```
设备名称：收音机 / 对讲机 / 太阳能充电器
外观描述：粗犷的工业设计，防摔防水
材质：工程塑料/金属
通讯范围：短波（数公里）/ 中继站（数十公里）
特性：
  - 可更换电池/太阳能充电
  - 手动调频
  - 部分型号可接收FM广播
  - 信号不稳定，有杂音
```

#### 2.2.8 近未来 — 数据终端【用户指定】

```
设备名称：数据终端 / Neural Link / 柔性屏终端
外观描述：柔性可折叠屏幕，或手腕/耳后植入设备
材质：纳米材料/生物相容涂层
通讯：量子网络/神经直连/6G
核心功能：
  1. 【全息地图】— 3D投影地图，AR叠加
  2. 【意识通讯录】— 直接神经交流，读取情绪
  3. 【神经群聊】— 思维共享，情感同步
  4. 【数据论坛】— 信息流，实时数据分析
  5. 【资讯过滤】— AI个性化推荐，意识干扰监测
  6. 【记忆存储】— 重要场景直接录制（神经信号）
  7. 【生物监测】— 身体状态、心率、脑波
  8. 【入侵检测】— 防护黑客入侵
```

#### 2.2.9 远未来太空歌剧 — 全息投影器

```

外观描述：悬浮光球/手环/戒指形态
材质：能量场/稀有金属

  1. 【星际地图】— 银河系范围，跃迁导航
```

## 四、UI/界面设计方案

### 4.1 界面设计原则

1. **时代一致性**：界面风格与设备形态、时代背景匹配
2. **功能可见性**：核心功能入口清晰，各时代统一
3. **沉浸感**：设备名称、按钮文案符合时代用语
4. **响应式**：适配桌面端和移动端

### 4.2 各时代UI风格

```
远古时代：
  - 配色：泥土色(#8B4513)、骨白(#F5F5DC)、炭黑(#2F2F2F)
  - 字体：粗糙手写体
  - 图标：简笔画/岩画风格
  - 按钮：石头质感

古代东方：
  - 配色：水墨色(#1C1C1C)、宣纸色(#F5F5DC)、朱砂色(#8B0000)
  - 字体：楷书/行书
  - 图标：山水/云纹/武侠元素
  - 按钮：玉石/木质质感

古代西方：
  - 配色：羊皮色(#F5DEB3)、橄榄色(#556B2F)、青铜色(#CD7F32)
  - 字体：古罗马体
  - 图标：希腊神话/罗马建筑元素
  - 按钮：羊皮纸/青铜质感

近代：
  - 配色：琥珀色(#FFBF00)、铜色(#B87333)、深棕(#3C2415)
  - 字体：打字机体
  - 图标：齿轮/机械/铁路元素
  - 按钮：木质+黄铜旋钮

现代：
  - 配色：现代扁平化/品牌色
  - 字体：系统默认
  - 图标：标准APP图标
  - 按钮：触屏标准

近未来：
  - 配色：霓虹色(#00FFFF/#FF00FF)、深黑(#0D0D0D)
  - 字体：等宽科技字体
  - 图标：全息/扫描线/电路
  - 按钮：虚空投影/悬浮

远未来：
  - 配色：星光色(#FFD700)、深空色(#0A0A23)
  - 字体：光纹体
  - 图标：星系/能量/全息
  - 按钮：能量场触控

后人类：
  - 配色：无固定，随意识变化
  - 字体：思维可视化
  - 图标：能量/粒子
  - 按钮：意念激活
```

### 5.4 入口点设计

**游戏内激活方式：**
1. **快捷键触发**：桌面端按 `M` 键，移动端点击悬浮按钮
2. **剧情触发**：特定场景自动弹出（如获得新设备）
3. **菜单触发**：从主菜单进入

**显示模式：**
1. **弹窗模式**：Modal 覆盖游戏画面
2. **分屏模式**：游戏画面与设备并排显示（可选设置）

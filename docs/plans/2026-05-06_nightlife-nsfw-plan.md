# 夜场/KTV生态 NSFW 模块详细方案

## 一、概述

### 1.1 模块定位

夜场/KTV生态系统是现代纪元 NSFW 模块的 P3 模块，聚焦于都市夜生活中的娱乐场所——KTV、酒吧、夜总会、会员制私人会所等，以及由此产生的酒水文化、社交博弈和暧昧氛围。

### 1.2 核心特色

**与传统游戏差异化：**
- 夜场是"释放压力的出口"——白天社会的规则在夜晚暂时失效
- 酒精降低防备——判断力下降，更容易产生亲密接触
- 消费即地位——开什么酒、坐什么位置代表身份
- 陪酒是"职业"——领队/妈咪管理的公主/少爷们
- 包厢是"私密社交空间"——比酒吧更封闭、更容易越界
- 醉酒状态创造机会——也可能创造危机

### 1.3 题材敏感度

```
本模块涉及：
✓ KTV/夜总会的陪酒文化
✓ 酒水消费与身份象征
✓ 包厢内的暧昧社交
✓ 醉酒状态与判断力下降
✓ 领队/妈咪管理系统
✓ 出台与不出台的边界
✓ 夜场的人脉网络
✓ 酒托/仙人跳等骗局

本模块不包括：
✗ 任何实际性行为的具体描写
✗ 未成年人相关敏感内容
✗ 真实夜场/品牌名称
```

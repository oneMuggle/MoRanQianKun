# 潜水/水上运动 NSFW 模块详细方案

## 一、概述

### 1.1 模块定位

潜水/水上运动系统是现代纪元 NSFW 模块的 P3 模块，聚焦于私密场景中的人际关系——潜水度假村、游艇派对、水上别墅等特殊环境下的暧昧氛围与信任建立。

### 1.2 核心特色

**与传统游戏差异化：**
- 水下是"隔绝的空间"——潜水时无法使用手机，与外界完全隔绝
- 潜水装备创造亲密距离——BCD（浮力调节装置）穿戴需要互相协助
- 水的环境放大感官——湿透的衣物、亲密的身体接触
- 游艇/岛屿是"法外之地"——私密性极高的社交空间
- 肾上腺素与亲密感——极限运动后的放松状态降低防备

### 1.3 题材敏感度

```
本模块涉及：
✓ 潜水度假村的私密环境
✓ 游艇派对上的暧昧社交
✓ 水上别墅的独处空间
✓ 潜水时的身体接触（BCD穿戴辅助、潜伴制度）
✓ 潜水教练与学员的特殊关系
✓ 海上派对的暧昧游戏
✓ 水下摄影的边界问题

本模块不包括：
✗ 任何实际性行为的具体描写
✗ 未成年人相关敏感内容
✗ 真实潜水度假村/游艇品牌名称
```

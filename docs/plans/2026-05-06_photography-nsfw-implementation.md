# 写真约拍 NSFW 模块实施方案

## 概述

将写真约拍 NSFW 子系统（P3，现代纪元）集成到墨色江湖游戏。模块覆盖从正规商业写真到私房约拍的完整生命周期——包括尺度递进、越界行为识别、照片泄露事件和模特保护机制。所有"摄影"都是文本叙事，不生成真实图片。

模块遵循现有 `campusNSFW` 和 `urbanDriverNSFW` 建立的架构模式：**types → engine → AI bridge → workflow → UI → wiring**。

## 架构决策

### 独立 State Mount

采用 `state.写真系统` 独立挂载点（不嵌套在 `state.校园系统` 下），跟随 `urbanDriverNSFW`（`state.都市网约车系统`）的先例。

**理由：**
- 写真约拍是现代纪元通用模块，不限于校园
- `urbanDriverNSFW` 已有独立 mount 先例
- 避免与校园特定的 `校园系统数据` 类型耦合

### 时代门控

通过 `时代配置ID` 前缀 `contemporary_` 判断是否启用，覆盖所有现代纪元子时代。

### Step 4: 引擎纯函数

**文件: `hooks/useGame/photographyNSFWEngine.ts`**（新文件，~350行）

实现所有纯计算函数（无副作用，无AI调用），遵循 `campusNSFWEngine.ts` 模式：

**常量配置（顶部）**：
- 场所风险映射：影棚=10, 外景=30, 酒店=60, 民宿=50, 个人住所=80, 野外=70
- 尺度风险映射：G级=10, PG-13=30, R级=60, NC-17=85, XXX=95
- 摄影师信誉风险映射：业界大佬=5, 资深=15, 普通=30, 有争议=60, 名声较差=80, 惯犯=95
- 交付风险映射：平台担保=5, 直接交付=30, 分期交付=50, 拖延交付=70, 拒绝交付=90

**核心函数**：
- `评估泄露风险(项目: 拍摄项目状态, 摄影师: 摄影师核心状态): number` — 加权平均风险值 0-100
- `计算尺度递进概率(摄影师越界倾向, 模特保护意识, 当前尺度): number` — 递进概率 0-1
- `判定是否尺度递进(概率: number): boolean` — 随机判定
- `判定越界行为(摄影师, 场所, 当前尺度): 越界行为类型 | null` — 基于权重的越界行为判定
- `计算摄影师口碑评分(摄影师): number` — 综合评分 0-100
- `筛选摄影师(摄影师列表, 模特保护意识): 摄影师核心状态[]` — 按口碑排序，过滤红线指标
- `判定泄露事件(风险值, 频率设置): { 触发: boolean, 类型: 泄露类型, 传播范围: 传播范围 }` — 概率泄露触发
- 工厂函数：`创建默认模特()`, `创建默认摄影师()`, `创建默认拍摄项目()`, `创建默认泄露事件()`

### 一、三层玩法 + 主层选择

写真模块提供三层玩法，玩家在新建游戏或设置中**选择一层作为主要玩法**。三层同时存在但权重和叙事焦点不同。

#### 第一层：经营管理层

**核心循环**：
```
设备投资 → 定价策略 → 接单排期 → 拍摄执行 → 后期修图 → 发布作品 → 积累客户 → 扩大规模
```

**关键系统**：
- **设备管理**：入门套装 → 升级机身 → 专业镜头 → 灯光系统 → 顶级棚。设备影响作品质量和可接的拍摄类型。
- **定价策略**：高价少单（精品路线）vs 低价多单（走量路线）。影响收入和模特群体。
- **档期管理**：每天/每周可接的拍摄数量有限。需要平衡时间和质量。
- **作品发布**：选择发布平台（微博/Instagram/摄影论坛/个人网站），不同平台吸引不同类型的模特和客户。
- **经营扩张**：从独立摄影师 → 租用影棚 → 雇助理 → 开工作室。

**主层体验**：类似模拟经营游戏。核心满足感来自"我的事业在成长"。

#### 第二层：人际关系层

**核心循环**：
```
发现模特 → 沟通了解 → 建立信任 → 合作拍摄 → 加深关系 → 口碑传播 → 推荐新模特
```

**关键系统**：
- **模特关系网**：每个模特有独立的好感度、信任度、合作满意度。
- **沟通系统**：拍摄前聊天了解需求，拍摄中互动，拍摄后反馈。好的沟通提升信任。
- **信任积累**：按时交付、尊重约定、保护模特隐私 → 信任提升。信任带来回头客和推荐。
- **口碑传播**：模特之间会互相推荐摄影师。口碑好的人脉网络自然扩张。
- **冲突处理**：模特不满意、投诉、要求重拍。处理方式影响关系走向。

**主层体验**：类似社交模拟游戏。核心满足感来自"我有一群信任我的人"。

#### 第三层：灰色地带层（NSFW核心）

**核心循环**：
```
试探边界 → 观察反应 → 决定是否越界 → 承担后果（正面或负面）→ 积累记录 → 触发事件
```

**关键系统**：
- 详见下方"NSFW深度设计"章节。

**主层体验**：道德选择与风险博弈。核心满足感来自"我在走钢丝，但我能控制"。

### 二、道德选择开关

灰色地带层的道德选择**默认关闭**。玩家需要主动开启，且开启后不可在当前存档中关闭（但可以在设置中查看当前状态）。

#### 开关层级

```
写真NSFW设置
├── 启用写真NSFW系统 [开关] ← 总开关
│
├── 主要玩法层 [选择器]
│   ├── 经营管理 (无道德选择)
│   ├── 人际关系 (无道德选择)
│   └── 灰色地带 ↓
│
├── 启用道德选择 [开关] ← 仅在主要层=灰色地带时可用
│   ├── 关闭 → 所有越界行为由AI决定，玩家被动接受叙事
│   └── 开启 → 玩家在关键节点面临主动决策
│
└── NSFW内容强度 [选择器]
    ├── 微暗 → 暗示性描述，不直接描写
    ├── 暧昧 → 适度描写，侧重心理和氛围
    └── 露骨 → 直接描写，不回避细节
```

#### 道德选择关闭时（被动模式）

- 摄影师的行为由AI根据其**隐藏动机**和**越界倾向**自动决定
- 玩家看到的是**结果叙事**——"这个摄影师做了什么"
- 类似于看故事，玩家不直接做道德决策
- 体验更接近"模特视角"的传统玩法

#### 道德选择开启时（主动模式）

- 在关键节点，系统给玩家提供**主动决策选项**
- 玩家的选择直接影响：摄影师状态、模特关系、风险积累
- 体验更接近"角色扮演"——玩家要为自己的选择负责
- 这是灰色地带的核心体验

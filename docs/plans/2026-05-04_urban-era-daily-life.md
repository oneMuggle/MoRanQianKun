# 都市纪元日常生活系统扩展方案

> 创建日期：2026-05-04
> 状态：待实施
> 关联纪元：当代都市 (`contemporary_urban`)

---

## 1. 现状分析

### 1.1 当前内容规模
- **开局场景**：3 个（都市霓虹、地下拳场、午夜街头）
- **角色原型**：3 个（科技新贵、地下拳王、白帽黑客）
- **写作范文**：2 篇
- **里模式**：已配置基础结构

### 1.2 不足之处
- 职业背景覆盖不完整（仅有都市职场类）
- 缺少配送出行、生活服务、蓝领技工、零售个体等职业入口
- 日常生活场景稀缺（无居家、无公共场所、无社交娱乐）
- 缺少都市特有的「天」系统（如日程安排、通勤、社交APP等）

---

## 2. 扩展方案

### 2.1 扩充开局场景
新增 7 个都市日常开局场景：

| ID | 名称 | 描述 | 适合职业 |
|----|------|------|----------|
| `urban_4` | 早高峰地铁 | 人潮汹涌的地铁站台，你是千万通勤族中的一员 | 白领、外卖骑手 |
| `urban_5` | 便利店夜班 | 24小时便利店的荧光灯下，形形色色的夜归人 | 店员、夜班工人 |
| `urban_6` | 工地旁早餐摊 | 清晨的工地门口，早餐摊的蒸汽弥漫 | 建筑工人、装修师傅 |
| `urban_7` | 写字楼大堂 | 玻璃幕墙的写字楼，你在前台等待面试 | 求职者、职场新人 |
| `urban_8` | 城中村握手楼 | 密密麻麻的城中村，阳光只能照进一线天 | 快递小哥、外卖骑手 |
| `urban_9` | 网吧包厢 | 烟雾缭绕的网吧，你在这里度过无数夜晚 | 自由职业者、创业者 |
| `urban_10` | 医院走廊 | 消毒水味的医院，你在家属等待区 | 战地医生、护士 |

### 2.2 扩充角色原型
新增 7 个都市角色原型：

| ID | 名称 | 描述 | 外貌特征 | 能力 |
|----|------|------|----------|------|
| `urban_delivery` | 外卖骑手 | 算法驱动下的配送大军一员 | 头盔、保温箱、电瓶车 | 城市导航、时间管理 |
| `urban_craftsman` | 蓝领技工 | 有一技之长的手艺人 | 工装、工具箱、粗糙的手 | 水电维修、机械 |
| `urban_retail` | 便利店店员 | 24小时便利店的守夜人 | 制服、围裙、疲惫的眼神 | 服务意识、观察力 |
| `urban_driver` | 网约车司机 | 城市的移动摄像头 | 驾驶座、手机支架、车载充电器 | 城市地理、社交 |
| `urban_coser` | 网红博主 | 自媒体时代的流量追逐者 | 补光灯、手机支架、精致的妆 | 内容创作、粉丝运营 |
| `urban_security` | 小区保安 | 守护一方平安的门神 | 制服、对讲机、警戒的眼神 | 观察、沟通、应急 |
| `urban_courier` | 快递小哥 | 最后一公里的奔跑者 | 三轮车、快递袋、汗湿的T恤 | 片区情报、攀爬 |

### 2.3 扩充背景数据

在 `data/backgrounds/modern.ts` 中，`contemporary_urban` 子纪元适配已有完整职业分类。需确认以下背景的天赋效果描述是否完整：

- 都市职场类：确认大厂员工、自媒体创业者、都市白领等天赋效果
- 配送出行类：确认网约车司机、外卖骑手、快递小哥等天赋效果
- 生活服务类：确认理发师、健身教练、推拿按摩师等天赋效果
- 蓝领技工类：确认装修师傅、汽修工、电工等天赋效果
- 零售个体类：确认便利店老板、夜市摊主、保险推销员等天赋效果

### 2.4 新增都市日常系统

#### 2.4.1 日程系统
- **工作日**：通勤 → 工作 → 午休 → 工作 → 下班 → 夜生活
- **休息日**：睡懒觉 → 逛商场 → 吃饭 → 看电影/娱乐 → 回家
- **特殊日程**：加班、应酬、约会、看病、购物

#### 2.4.2 通勤系统
- 地铁/公交：随机事件（让座、冲突、艳遇）
- 骑车/步行：天气影响、体力消耗
- 网约车：乘客/司机身份切换
- 开车：堵车焦虑、停车难题

#### 2.4.3 社交APP系统
- 社交软件：附近的人、摇一摇、漂流瓶
- 外卖平台：骑手端/用户端
- 打车平台：司机端/乘客端
- 职场社交：脉脉、领英

### 模块一：高端会所 / SPA 技师系统

**文件路径：** `models/contemporary/eliteClub.ts`

**现实参考：** 私人SPA会所、会员制高端会所、技师与VIP客户的暧昧关系

**核心类型：**

```typescript
// 服务等级
export type 客户等级 = '普通会员' | '白银VIP' | '黄金VIP' | '白金VIP' | '黑卡会员';

// 技师评级
export type 技师评级 = '初级' | '中级' | '金牌' | '红牌' | '传奇';

// 特殊服务类型
export type 特殊服务 = '精油按摩' | '热石理疗' | '泰国按摩' | '私人定制' | '神秘项目';

// 会所包厢类型
export type 包厢类型 = '普通包厢' | 'VIP包厢' | '总统包厢' | '露天池畔' | '私人别墅';
```

**核心机制：**

- 服务等级系统：普通会员 → 白金VIP → 黑卡会员（解锁内容逐级提升）
- 技师评级与客户指定：客户可指定特定技师，等级越高可指定技师级别越高
- 私密包厢场景触发：根据包厢类型和客户等级触发不同深度场景
- 客户「特殊要求」系统：白金以上客户可提出特殊要求，技师可接受/拒绝
- 小费机制与客户忠诚度：小费影响技师收入和服务态度
- 会所内部潜规则：领队/妈咪管理、技师之间的竞争与合作
- 技师培训与升级：通过服务积累经验提升评级

**关键状态：**

```typescript
export interface 会所技师状态 {
  技师ID: string;
  当前评级: 技师评级;
  工作经验: number;
  客户评分: number;

  服务中心情状态: '愉快' | '平常' | '压抑' | '抗拒';
  隐秘日记: string[];  // 记录不公开的服务经历
}

export interface VIP客户状态 {
  客户ID: string;
  会员等级: 客户等级;
  账户余额: number;
  累计消费: number;
  专属技师ID?: string;
  特殊权限: string[];
  入会时间: string;
  信用记录: '优良' | '正常' | '欠佳' | '黑名单';
}
```

**场景示例：**

- 黑卡会员的深夜指定服务
- 技师在包厢中的艰难选择
- 私人别墅派对：高端客户的隐秘聚会
- 技师评级考核：金牌升级赛
- 会所妈咪的秘密账本

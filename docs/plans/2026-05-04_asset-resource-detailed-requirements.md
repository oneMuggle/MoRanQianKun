| 素材类型 | 推荐来源 | 注意事项 |
|----------|----------|----------|
| 场景图 | NovelAI / ComfyUI / Midjourney | 统一风格前缀 |

## 八、风险与依赖

| 风险 | 等级 | 应对 |
|------|------|------|
| AI 生成图片风格不一致 | 高 | 统一 style preset + negative prompt + 后处理 |
| BGM 版权问题 | 高 | 严格使用 CC0/CC-BY，记录 attribution |
| CDN 存储/带宽成本 | 中 | 图片用 WebP 压缩，音频 128kbps MP3 |
| 字体体积影响加载 | 中 | WOFF2 + unicode subset + font-display: swap |
| MiniMax API 生成失败 | 中 | 降级到 NovelAI/ComfyUI 手动生成 |
| 42 个时代表现差异大 | 低 | 分批上线，优先古代+现代高频时代 |

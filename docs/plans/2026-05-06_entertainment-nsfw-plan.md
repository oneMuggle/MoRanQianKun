# 偶像圈/演艺圈 NSFW 模块详细方案

## 一、概述

### 1.1 模块定位

偶像圈/演艺圈系统是现代纪元 NSFW 模块的 P1 核心模块，聚焦于娱乐产业中的权力不对等关系、粉丝-偶像的畸形亲密、以及行业潜规则。

### 1.2 核心特色

**与传统游戏差异化：**
- 偶像-粉丝关系是"被构建的亲密"——表面是粉丝暗恋偶像，实际是偶像团队精心设计的情感消费
- 潜规则是权力不对等的结构性产物——不是偶然而是系统性
- 私生饭是"越界亲密"的极端形态——从追到跟踪到犯罪
- 偶像本身是"被凝视的商品"——完美人设背后是真实人格的压抑

### 1.3 题材敏感度

```
本模块涉及：
✓ 娱乐圈潜规则（性暗示、条件交换）
✓ 私生饭行为（跟踪、骚扰、隐私侵犯）
✓ 选秀黑幕（不公、淘汰交易）
✓ 偶像人设崩塌（恋情、私生活曝光）
✓ 粉丝文化（控评、anti、网络暴力）

本模块不包括：
✗ 任何实际性行为的具体描写
✗ 未成年人相关敏感内容
✗ 真实艺人/真实公司名称
```

## 四、核心机制详解

### 4.1 练习生培育机制

**培训阶段：**

```
新人培训（0-6个月）
├── 基础唱跳训练
├── 形象管理
├── 媒体应对培训
└── 心理建设（服从性培养）

中期培训（6-18个月）
├── 专项能力提升
├── 舞台经验积累
├── 人设确定
└── 开始参与选秀/小活动

高级培训（18-36个月）
├── 出道准备
├── 粉丝运营
├── 资源积累
└── 出道位谈判
```

**能力评估系统：**

| 能力维度 | 评估指标 | 提升方式 |
|----------|----------|----------|
| 唱功 | 音域、音色、稳定性 | 声乐课、实战 |
| 舞蹈 | 协调性、表现力、难度 | 舞蹈课、练习 |
| 表现力 | 表情管理、镜头感 | 表情课、拍摄练习 |
| 外貌 | 五官、身材、辨识度 | 形象管理、医美 |
| 粉丝缘 | 亲和力、人格魅力 | 粉丝互动、综艺 |

### 4.2 潜规则应对系统

**潜规则触发机制：**

```
触发条件：
├── 艺人处于弱势地位（新人、过气）
├── 资源方有权力（导演、投资人、高层）
├── 私下场合（酒局、包厢、私下约见）
└── 艺人主动或被动赴约

潜规则类型演化路径：
暗示 → 试探 → 条件交换 → 实施 → 持续或终止
```

**应对方式分支：**

| 应对 | 结果（短期） | 结果（长期） | 适用情况 |
|------|--------------|--------------|----------|
| 接受 | 获得资源 | 持续被索取、心理健康下降 | 急需资源、无其他选择 |
| 拒绝 | 失去机会 | 可能被冷藏或被迫退圈 | 有底气、有退路 |
| 周旋 | 保持关系不得罪 | 可能被视为"装" | 高情商、有技巧 |
| 反抗 | 当下受损 | 可能反转、可能彻底被封杀 | 有证据、有靠山 |
| 交易 | 获得对等交换 | 双方达成默契 | 谈判能力强 |
| 转嫁 | 自身脱身 | 道德谴责、可能暴露 | 不推荐 |

### 4.3 粉丝关系演化系统

**粉丝-偶像关系阶段：**

```
第一阶段：路人 → 粉丝
├── 接触作品（歌曲、影视、综艺）
├── 情感投入开始
└── 关注社交媒体

第二阶段：粉丝 → 核心粉丝
├── 投入时间/金钱
├── 加入粉丝群体
├── 参与打投、控评
└── 自我认同"我是XXX粉丝"

第三阶段：核心粉丝 → 大粉
├── 影响力扩大
├── 组织粉丝活动
├── 与艺人团队有联系
└── 获得小福利（提前知道行程等）
```

# 小说分解附加功能

> **Status:** 进行中 | **当前阶段:** 阶段 7 / 质量与注入一致性收尾 | **最近更新:** 2026-03-30

## 目标

为同人模式新增"小说分解附加功能"，支持长篇小说分段拆分、结构化总结、树状注入、断点续跑、后台拆分、JSON 分享与运行时注入查看。

## 总体范围

- 首页新增小说分解独立工作台入口
- 设置内仅保留小说分解独立 API 配置
- 支持分段拆分、进度保存、断点恢复
- 支持后台拆分与边玩边拆
- 支持主剧情/规划分析/世界演变三层注入
- 支持树状多条目管理与注入查看
- 支持 JSON 导入导出与分享

## 关键设计原则

- 主剧情只吃轻量概括，不直接吞详细树
- 已完成分段立即可注入，未完成分段不阻塞游玩
- 所有长任务必须可恢复，避免一次失败全盘重来
- 结构化结果优先，原始文本只做辅助展示

## 风险与注意事项

- 主剧情注入必须严格控量，避免上下文暴涨
- 后台拆分需要避免阻塞正常主流程发送
- 任务进度与分解结果需要分开持久化，避免相互污染
- JSON 分享格式需要从一开始就考虑版本号
- 当前 AI 分解失败时会回退到本地启发式摘要

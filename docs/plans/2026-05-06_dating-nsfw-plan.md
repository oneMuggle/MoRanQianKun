# 婚恋相亲 NSFW 模块详细方案

## 一、概述

### 1.1 模块定位

婚恋相亲系统是现代纪元 NSFW 模块的 P3 模块，聚焦于现代都市中的婚恋现象——相亲角、婚介所、约会软件、以及由此产生的情感欺骗、物质博弈和身份焦虑。

### 1.2 核心特色

**与传统游戏差异化：**
- 相亲是"明码标价的交易"——年龄、学历、收入、房产、颜值都是筹码
- 婚姻是"两个家庭的结合"——背后是两个家庭的博弈
- 剩男剩女是"社会焦虑"——年龄焦虑、性别焦虑、阶层焦虑
- 婚恋平台是"流量游戏"——杀猪盘、酒托、骗婚
- 婚礼是"表演"——排场、彩礼、嫁妆的展示

### 1.3 题材敏感度

```
本模块涉及：
✓ 相亲市场的物质化
✓ 年龄焦虑与性别焦虑
✓ 婚介所骗局
✓ 约会软件暗面（酒托、杀猪盘）
✓ 婚礼的排场与压力
✓ 彩礼嫁妆纠纷
✓ 婚后出轨/家庭暴力
✓ 离婚财产分割

本模块不包括：
✗ 未成年人相关敏感内容
✗ 真实婚恋平台/品牌名称
✗ 实际性行为的具体描写
```

```typescript
const 成功率评估 = (男方条件, 女方条件) => {
  const 男方总分 = 计算加权总分(男方条件, 权重.男性视角);
  const 女方总分 = 计算加权总分(女方条件, 权重.女性视角);

  if (Math.abs(男方总分 - 女方总分) > 30) {

  // 基础成功率

  if (男方物质 > 女方物质 * 2) 成功率 += 0.1;

  return Math.max(0.1, Math.min(0.9, 成功率));

```

```
   ├── 条件筛选（年龄、学历、收入）

   └── 长相筛选（照片）

2. 接触方式
   ├── 直接搭话
   ├── 留联系方式

   ├── "你孩子多大了？"
   ├── "什么学历？什么工作？"
   ├── "有房有车吗？"

- 条件好的被疯抢
- 条件差的自嘲

```

```
[婚介所 - 工作日]
```

```
↓ 职场权力不对等下的感情
```

```
↓ 企业家相亲会
↓ 门槛极高的VIP俱乐部

```

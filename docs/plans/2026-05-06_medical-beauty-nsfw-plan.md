# 整形/医美行业 NSFW 模块详细方案

## 一、概述

### 1.1 模块定位

整形/医美行业系统是现代纪元 NSFW 模块的 P2 模块，聚焦于现代社会的医美产业——从整形手术到皮肤管理，以及由此产生的审美焦虑、身体物化、医美贷陷阱和手术风险。

### 1.2 核心特色

**与传统游戏差异化：**
- 医美是"身体的资本化"——把身体当作可以改造的资产
- 审美焦虑是社会性的——白瘦幼、锥子脸、A4腰的流行标准
- 医美贷是"债务陷阱"——整形贷让年轻人背负债务
- 手术风险是真实的——失败案例、并发症、死亡率
- 医美是"成瘾"的——一次手术后会产生更多不满

### 1.3 题材敏感度

```
本模块涉及：
✓ 整形手术（割双眼皮、隆胸、抽脂等）
✓ 皮肤管理/微整形（玻尿酸、水光针等）
✓ 医美贷/分期付款
✓ 审美焦虑与身体形象
✓ 医美失败与维权
✓ 医美上瘾与心理问题
✓ 医美直播/案例分享
✓ 医美中介/渠道

本模块不包括：
✗ 未成年人相关敏感内容
✗ 真实医美机构/品牌名称
✗ 实际手术过程的具体描写
```

## 五、场景系统

### 5.1 咨询场景

**场景一：面诊设计**

```
触发条件：
- 玩家去医美机构咨询
- 顾问/咨询师接待

场景描述：
[医美机构 - 下午]
咨询师热情接待...

面诊流程：
1. 外观评估
   ├── 咨询师观察你的脸
   ├── 指出"问题"
   ├── 放大你的焦虑
   └── "你的鼻子如果再高一点..."

2. 方案设计
   ├── 推荐各种项目
   ├── 多个项目打包
   ├── 看起来很划算
   └── 其实很多没必要做

3. 价格谈判
   ├── 报价很高
   ├── 假装申请优惠
   ├── "今天交定金再减..."
   └── 逼单手段

4. 支付方式
   ├── 全款or分期
   ├── 分期利息多少
   ├── 引导贷款
   └── "0利息0手续费"

关键选择：
- 是否相信咨询师的判断
- 要不要做推荐的项目
- 分期还是全款
- 是否签合同
```

**场景二：医美贷陷阱**

```
触发条件：
- 玩家钱不够但想做手术
- 被推荐医美贷

场景描述：
[医美机构 - 贷款签约]
咨询师推荐你分期...

贷款套路：
1. 诱导分期
   "你本来可以做更好的医生，
   分期的话每月只要500..."

2. 平台选择
   ├── 推荐某个贷款平台
   ├── 声称正规低息
   └── 其实利息很高

3. 合同签署
   ├── 合同密密麻麻
   ├── 各种费用条款
   ├── 违约金很高
   └── 当场签字没细看

4. 贷后问题
   ├── 实际还款比预估多
   ├── 逾期催收电话
   ├── 影响征信
   └── 以贷养贷越欠越多
```

```

- 被贴上"整容女"标签
- 男朋友/老公发现
- 工作受影响
- 心理创伤

维权难度：
- 难以证明来源
- 机构推卸
- 网络传播快
- 删除难以彻底
```

### 5.3 医美博主场景

**场景五：医美直播**

```
触发条件：
- 玩家关注医美博主
- 博主推荐项目/医院

场景描述：
[直播间 - 晚间]
医美博主在直播...

博主类型：
1. 真实分享型
   ├── 分享自己的经历
   ├── 客观分析风险
   └── 推荐正规机构

2. 商业推广型
   ├── 拿医院回扣
   ├── 夸大效果淡化风险
   └── 推的医院不一定好

3. 虚假案例型
   ├── 案例照片造假
   ├── 盗用他人照片
   └── 完全是托

4. 医托型
   ├── 以网友身份推荐
   ├── 实际上是托
   └── 引流到特定医院

风险点：
- 博主可能没做过
- 医院给回扣
- 案例可能造假
- 效果无法保证
```

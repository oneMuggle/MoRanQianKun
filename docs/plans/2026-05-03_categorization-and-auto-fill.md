## 目标

1. **分类筛选**：为天赋和背景增加分类标签，在新手向导 UI 中按分类浏览（特别针对现代纪元 200+ 条目）

## 风险与缓解

| 风险 | 级别 | 缓解措施 |
|------|------|---------|
| `presets.ts` 文件膨胀（当前 ~956 行） | 中 | 分类字段为短字符串，增量仅 ~3-5KB |
| 自动填充覆盖用户已有选择 | 中 | 仅填充空位，绝不替换已选项 |
| 背景→天赋/气运映射名称不匹配 | 中 | 精确名称匹配 + 回退到已有查找函数 |
| 移动端 UI 结构不同 | 低 | 先检查移动端组件再镜像实现 |
| 分类值在背景和天赋间不一致 | 低 | 使用常量定义分类值，单源维护 |

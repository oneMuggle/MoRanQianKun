# 野外/极限运动 NSFW 模块详细方案

## 一、概述

### 1.1 模块定位

野外/极限运动系统是现代纪元 NSFW 模块的 P4 模块，聚焦于户外运动场景中涉及的边界模糊行为——从正规户外探险到野外生存、极限挑战中产生的信任滥用、偷拍传播、越界行为等问题。

### 1.2 核心特色

**与传统游戏差异化：**
- 野外环境是"法外之地"——远离城市、信号受阻、难以求援
- 极限运动存在天然权力不对等——教练/领队掌握参与者安全
- 私密性极高——野外/高山/水下等环境难以被外界发现
- 记录与泄露风险——运动相机/无人机等设备的广泛使用
- 肾上腺素与边界模糊——极限状态下的判断力下降

### 1.3 题材敏感度

```
本模块涉及：
✓ 野外/极限场景的尺度边界（运动 vs 暴露/挑逗）
✓ 教练/领队与参与者的关系（指导/暧昧/越界）
✓ 偷拍与未经同意的传播（运动相机/无人机）
✓ 极限挑战中的越界行为（强迫/诱导）
✓ 运动内容的泄露与传播
✓ 野外环境中的性骚扰/侵犯

本模块不包括：
✗ 任何实际性行为的具体描写
✗ 未成年人相关敏感内容
✗ 真实户外品牌/俱乐部名称
✗ 真实野外环境的具体定位信息
```

### 4.2 极限运动尺度递进机制

**尺度分级定义：**

| 尺度等级 | 服装要求 | 拍摄范围 | 氛围 | 典型场景 |
|----------|----------|----------|------|----------|
| 正常 | 专业运动服 | 运动过程 | 专注挑战 | 正常登山/徒步 |
| PG-13 | 可有少量暴露 | 运动+休息 | 轻松愉快 | 溪流玩水/露营休息 |
| R级 | 泳装/湿衣 | 局部特写 | 性感氛围 | 瀑布下/海边 |
| NC-17 | 内衣/情趣 | 身体曲线 | 情色暗示 | 深夜帐篷/篝火旁 |
| XXX | 全裸/露点 | 明确性暗示 | 明确性 | 无人湖区/山洞 |

**递进触发场景：**

```
活动前约定：
├── 约定活动：徒步穿越
├── 约定服装：专业运动服
└── 约定内容：正常户外活动记录

活动中递进触发：
├── 教练说"这里风景好，脱掉外套拍几张"
├── "你的身材很适合穿比基尼，下次可以带"
├── "晚上篝火旁大家都很开放，放松点"
├── "这边没人，过来拍点私人的"
├── "这个姿势很美，把衣服脱了吧"
└── 参与者的应对选择

递进结果：
├── 接受 → 尺度升级，边界进一步模糊
├── 拒绝 → 可能被孤立或被丢在危险环境
├── 安全词 → 提前结束活动，可能无法获得救援
└── 反抗 → 可能遭到报复或威胁
```

## 五、场景系统

### 5.1 正规户外活动场景

```
场景：商业俱乐部一日徒步
参与人数：10-20人
教练比例：1:5
场所：国家公园/正规景区
住宿：无需过夜

【正常流程】
报名 → 缴费用 → 集合出发 → 正常徒步 → 途中休息拍照 → 下山返程

【风险点】
⚠️ 商业活动的正规外衣下可能隐藏不当拍摄
⚠️ 集体活动也存在落单被侵犯风险
⚠️ 照片发布到社交媒体可能未经同意
```

### 5.2 野外露营场景

```
场景：野外两日一夜露营
参与人数：5-10人
教练比例：1:3
场所：野山/无人区
住宿：帐篷

【正常流程】
集合 → 徒步进山 → 扎营 → 晚餐 → 篝火 → 休息 → 早餐 → 拔营 → 返程

【风险点】
⚠️ 篝火/酒精氛围降低防备
⚠️ 夜间帐篷内隐私被侵犯
⚠️ 混帐安排增加风险
⚠️ 夜间偏僻处落单危险
⚠️ 第二天意识模糊时的记忆缺失
```

### 5.3 极限挑战拍摄场景

```
场景：野外生存挑战拍摄
参与人数：2-5人
性质：配合拍摄活动
场所：完全无人区
住宿：露天/简易

【正常流程】
签约 → 设备准备 → 进入无人区 → 挑战任务 → 记录素材 → 完成任务

【风险点】
⚠️ 完全无信号无法求援
⚠️ 摄像机24小时跟拍隐私
⚠️ 极端环境下的服从性增加
⚠️ 合同条款可能存在陷阱
⚠️ 内容可能被用于其他用途
```

### 5.4 水上运动场景

```
场景：潜水/溯溪活动
参与人数：3-8人
教练比例：1:2
场所：湖泊/溪流/海边
住宿：民宿/帐篷

【正常流程】
集合 → 装备讲解 → 热身 → 入水 → 水下活动 → 上岸休息 → 返程

【风险点】
⚠️ 湿衣/泳装状态下身体暴露
⚠️ 水下拍摄难以察觉
⚠️ 浮力衣装备可能被利用
⚠️ 水下不当接触难以反抗
⚠️ 出水后更换衣物时的暴露
```

### 5.5 野外探险偷拍场景

```
场景：以探险为名的偷拍

性质：伪装成正常活动

⚠️ 通过网络/社交媒体招募
⚠️ 费用异常低廉或全免
⚠️ 对参与者外表有要求

⚠️ 禁止带其他人同行
```

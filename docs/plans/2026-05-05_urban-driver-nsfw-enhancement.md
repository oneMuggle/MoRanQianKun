# 都市纪元 - 网约车司机 NSFW 强化方案

## 背景与目标

### 背景
当前项目已有一套成熟的校园 NSFW 深化系统（v1.4），包含欲望状态机、关系轨道、露出系统、SM/支配服从、桌游社交、校园祭等子系统。都市纪元（`contemporary_urban`）的里模式（里都市）已有基础 NSFW 框架（职场秘事、权色交易等），但缺少针对**网约车司机**这一具体职业的专属 NSFW 内容。

### 目标
为都市纪元的网约车司机职业路径增加深度 NSFW 强化，核心场景包括：
1. **醉酒女乘客** — 深夜接到醉酒状态的女性乘客，在密闭车厢内产生暧昧/亲密互动
2. **车上饮料下药** — 乘客或司机在车内饮料中被下药导致意识模糊、判断力下降的场景
3. **其他衍生场景** — 深夜独乘、后座暗示、停车场秘密、拼车暧昧、行车记录仪等

### 设计原则
- **复用校园 NSFW 的架构模式**（模型层 → 引擎层 → 提示词层 → UI 层），但领域语义独立
- **独立于校园系统**，不共享类型或状态，避免语义混淆
- **可配置化**，所有子系统均有独立开关和强度控制
- **后果驱动**，每个 NSFW 场景都伴随可能的后果（平台投诉、行车记录仪曝光、药物后遗症等）

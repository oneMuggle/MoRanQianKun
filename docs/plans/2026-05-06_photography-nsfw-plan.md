# 写真约拍 NSFW 模块详细方案

## 一、概述

### 1.1 模块定位

写真约拍系统是现代纪元 NSFW 模块的 P3 模块，聚焦于摄影行业的边界模糊地带——从正规商业写真到私密私房摄影之间的灰色地带，以及由此产生的权力不对等、信任滥用和内容泄露问题。

### 1.2 核心特色

**与传统游戏差异化：**
- 写真拍摄是"被凝视的身体"——镜头前的表演与真实的自我之间的张力
- 摄影师与模特存在天然权力不对等——摄影师掌握成片质量和模特形象
- 尺度递进是渐进式的——从"艺术"到"擦边"到"明确NSFW"的边界游走
- 照片/视频一旦拍摄就存在泄露风险——数字痕迹无法完全抹除

### 1.3 题材敏感度

```
本模块涉及：
✓ 私房摄影的尺度边界（艺术 vs 色情）
✓ 摄影师与模特的关系（专业/暧昧/越界）
✓ 照片泄露与传播（私密照变公开）
✓ 模特行业的潜规则（以拍换机会）
✓ 偷拍与未经同意的传播
✓ 拍摄过程中的越界行为

本模块不包括：
✗ 任何实际性行为的具体描写
✗ 未成年人相关敏感内容
✗ 真实摄影师/摄影工作室名称
```

## 四、核心机制详解

### 4.1 拍摄尺度递进机制

**尺度分级定义：**

| 尺度等级 | 服装要求 | 拍摄角度 | 氛围 | 传播范围 |
|----------|----------|----------|------|----------|
| G级 | 日常服装 | 正常 | 清新自然 | 全年龄段可看 |
| PG-13 | 泳装/睡衣 | 正常 | 清新/时尚 | 青少年以上 |
| R级 | 内衣/透明 | 可有特写 | 性感暧昧 | 成人内容需标注 |
| NC-17 | 情趣内衣/局部 | 强调身体曲线 | 情色暗示 | 仅成人平台 |
| XXX | 全裸/露点 | 明确性暗示 | 明确性 | 色情网站 |

**尺度递进触发场景：**

```
拍摄前约定：
├── 约定尺度：G级
├── 约定服装：日常便装
└── 约定氛围：清新自然

拍摄中递进触发：
├── 摄影师说"这套衣服拍出来效果不好，换一套试试？"
├── "这个角度更显身材"
├── "放松一点，可以靠在床上"
├── "试试这个姿势，很美的"
└── 模特的应对选择

递进结果：
├── 接受 → 尺度升级，边界进一步模糊
├── 拒绝 → 摄影师可能不满，或换其他方式试探
└── 安全词 → 终止拍摄，可能引发后续问题
```

```
├── 报价合理
├── 没有投诉记录
└── 愿意签合同/付定金

B. 普通摄影师（评分 60-80）
├── 作品质量一般
├── 基本没有投诉
├── 报价正常
└── 合作过但无特别印象

C. 有争议摄影师（评分 40-60）
├── 传过投诉/绯闻
├── 约拍条件过于优厚（可能有陷阱）
├── 作品质量一般但报价很高
└── 需要谨慎

D. 危险摄影师（评分 < 40）
├── 多次被投诉
├── 偷拍/越界记录
├── 要求模糊（不签合同、不付定金）
├── 场所选择可疑
└── 圈内警告过
```

### 4.4 照片交付与泄露机制

**交付流程：**

```
拍摄完成
├── 当天或次日：原片初步筛选
│   ├── 摄影师删选（去掉明显废片）
│   └── 原片发给模特确认
│
├── 模特确认：选择可以精修的
│   ├── 精选张数通常为拍摄张数的 10-20%
│   └── 不适合公开的照片注明"仅本人留存"
│
├── 后期修图
│   ├── 精修张数 = 精选张数的 50-80%
│   └── 修图风格确认
│
└── 交付成片
    ├── 精修原图交付（通常有水印）
    ├── 压缩图/九宫格发布用图
    └── 模特确认满意后结款
```

### 4.5 模特保护机制

**拍摄前保护：**

```
1. 信息核实
   ├── 查看摄影师真实姓名、联系方式
   ├── 核实作品是否本人拍摄（可通过识图）
   ├── 查询摄影师口碑（模特群、论坛）
   └── 要求查看身份证/营业执照

2. 合同签订
   ├── 明确拍摄内容、尺度、用途
   ├── 约定原片/精修片交付时间
   ├── 约定保密条款（照片不得外传）
   ├── 约定违约责任
   └── 约定安全词

3. 告知第三人
   ├── 将拍摄信息告知朋友/家人
   ├── 约定时间联系确认安全
   ├── 发送摄影师联系方式
   └── 发送拍摄地点定位

4. 安全准备
   ├── 选择公共场所/正规影棚
   ├── 不喝对方提供的饮料
   ├── 自备换装空间
   └── 携带防狼喷雾/报警器
```

**拍摄中保护：**


**拍摄后保护：**

```
1. 照片确认

   ├── 确认精修范围

3. 后续跟进
   ├── 确认照片用途
   ├── 定期搜索自己照片（防止泄露）
   ├── 发现问题立即处理
   └── 建立长期信任摄影师名单
```

```
模特：（边界已经完全模糊）

最终状态：
├── 模特事后后悔
```

**场景六：越界拒绝**

```
触发条件：
- 摄影师提出明确越界要求
- 模特选择拒绝/使用安全词

场景描述：
[拍摄过程中]
摄影师要求模特做超出约定的动作...

可选回应：

选项A：直接拒绝
「不行，这个超出我们的约定了」
→ 摄影师可能不满，拍摄气氛变差
→ 模特坚持底线

选项B：使用安全词
「[安全词]，我需要暂停一下」
→ 拍摄终止
→ 可能被要求赔偿/被扣照片

选项C：委婉周旋
「这个我不太擅长，换一个吧」
→ 保持表面和平
→ 但可能被视为"可以继续试探"

选项D：录音留证
（偷偷按下录音）
「您刚才说什么？我没听清」
→ 收集证据
→ 后续可作为投诉/报警依据
```

```
├── "怎么会这样？"
├── "是谁泄露的？"
└── "我该怎么办？"

事件演化：
```

# 宠物经济 NSFW 模块详细方案

## 一、概述

### 1.1 模块定位

宠物经济系统是现代纪元 NSFW 模块的 P3 模块，聚焦于现代都市中的宠物服务产业——宠物美容、宠物酒店、宠物医疗、宠物租赁、以及由此产生的情感消费与身份认同。

### 1.2 核心特色

**与传统游戏差异化：**
- 宠物是"家庭成员"——现代人将宠物视为情感寄托
- 宠物服务是"消费升级"——从基础喂养到全方位服务
- 宠物博主是"流量密码"——宠物网红经济兴起
- 宠物租赁是"共享经济"——共享宠物的边界问题
- 宠物比赛是"阶层象征"——纯种血统与展示文化
- 宠物医疗是"暴利行业"——过度医疗与情感消费

### 1.3 题材敏感度

```
本模块涉及：
✓ 宠物美容/宠物酒店服务
✓ 宠物医疗与整形
✓ 宠物繁育与交易
✓ 宠物租赁/共享经济
✓ 宠物博主与宠物网红
✓ 宠物比赛与血统认证
✓ 宠物保险与金融服务
✓ 宠物咖啡馆/撸猫馆

本模块不包括：
✗ 任何动物虐待的详细描写
✗ 未成年人与宠物的敏感场景
✗ 真实宠物店/品牌名称
```

## 五、场景系统

### 5.1 宠物购买场景

**场景一：星期狗陷阱**

```
触发条件：
- 玩家从非正规渠道购买宠物
- 可能买到病宠

场景描述：
[路边/网上 - 交易中]
看到一只可爱的宠物...

交易过程：
1. 选宠
   ├── 看外观（可能染过色/打过血清）
   ├── 问价格（明显低于市场价）
   └── 看证书（可能是假的）

2. 交易
   ├── 付款（可能不支持退款）
   └── 带走宠物

3. 回家后
   ├── 前几天精神好（血清作用）
   ├── 第三天开始萎靡
   └── 第七天发病/死亡

选择点：
- 是否去正规渠道购买
- 是否要求健康证明
- 是否签订购买合同
- 出现问题后如何维权
```

**场景二：正规猫舍购买**

```
触发条件：
- 玩家决定从正规猫舍购买
- 有血统证书和保障

场景描述：
[正规猫舍 - 下午]
参观猫舍并选择猫咪...

猫舍体验：
1. 环境考察
   ├── 猫舍环境（干净、宽敞）
   ├── 猫咪状态（健康、亲人）
   └── 繁育人态度（专业、负责）

2. 选猫
   ├── 了解血统背景
   ├── 看父母照片/视频
   ├── 选择心仪的小猫
   └── 确认价格（含绝育/疫苗）

3. 后续
   ├── 签订合同
   ├── 提供血统证书
   ├── 提供疫苗记录
   └── 售后保障说明
```

### 5.2 宠物医疗场景

**场景三：医疗过度**

```
触发条件：
- 宠物生病
- 面对宠物医院的过度医疗建议

场景描述：
[宠物医院 - 诊断中]
医生给出治疗方案...

医生可能过度医疗：
1. 套餐检查
   ├── 医生："先做个全身检查吧"
   ├── 实际：普通感冒不需要全身检查
   └── 费用：2000元

2. 高级用药
   ├── 医生："用进口药吧，效果好"
   ├── 实际：国产药同样有效
   └── 费用：500元 vs 50元

3. 不必要手术
   ├── 医生："建议做手术根治"
   ├── 实际：保守治疗也能好
   └── 费用：5000元 vs 500元

选择点：
- 是否接受全套检查
- 是否选择高价方案
- 是否要求解释必要性
- 是否换一家医院
```

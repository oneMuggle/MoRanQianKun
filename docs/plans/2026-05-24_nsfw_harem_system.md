# 实施计划：NSFW 后宫系统深化（多人 NSFW 专属玩法）

## 概述

基于现有系统基础，构建**多人 NSFW 玩法层**，让玩家与多名后宫成员的亲密/NSFW 互动不再是独立的单线事件，而是有角色关系、资源管理、场景演化、风险后果的多角色互动游戏。

## 现有系统盘点（多人 NSFW 可直接复用的地基）

| 现有系统 | 已有能力 | 可直接复用的部分 |
|----------|----------|-----------------|
| `multiCharacterNSFW.ts` | 嫉妒值、冲突判定(阈值70)、4种平衡策略、"后宫之主"成就 | `嫉妒冲突判定`、`计算嫉妒变化`、`多角关系成就判定` |
| `nsfwCore/resources.ts` | 精力值/声誉/亲密代币/风险预算 + 操作函数 | `消耗精力`、`消费亲密代币`、`修改声誉` |
| `nsfwCore/shopSystem.ts` | 商店道具系统 | 道具购买/使用框架，扩展后宫专属商品 |
| `intimacy.ts` | 5级亲密 + 双修属性奖励(1-3随机属性) | `生成双修奖励`、`计算双修收益` |
| `crossSystemLinker.ts` | 事件总线 + 联动规则 | 事件生成/处理框架 |
| `nsfw.ts` | 3档 NSFW 叙事约束(点到为止/适度展开/完全展开) | `自动选择叙事约束`、时代自适应 |
| `energyManagement.ts` | 精力消耗/恢复/规划 | `回合结束精力处理`、`精力不足惩罚` |

### 玩法 1：双修大会（Multi-Person Dual Cultivation）

**核心概念**：玩家同时与多名后宫成员进行双修，人数越多、成员间关系越好，收益越高但风险也越大。

#### 双修大会模式

```

│ 双修大会 - 选择参与成员                      │
├──────────────────────────────────────────────┤
│ [✓] 王语嫣 (正室·Lv5)  和谐度: ████░ 85     │

│                                              │
│ 参与人数: 3人 | 平均和谐度: 78               │
│ 预计消耗: 45 精力 | 风险: 中等                │
│                                              │
│ 预计收益: +3~5 随机属性 + 专属组合效果        │
```

#### 双修大会规则

```typescript

  参与成员: {
```

#### 收益计算

```
基础收益 = 参与人数 × (1~3 随机属性)

和谐度加成:
  平均和谐度 >= 80: 额外 +2 属性, 所有属性 +1 上限
  平均和谐度 >= 60: 额外 +1 属性
  平均和谐度 < 40: 属性 -1 (最低1)

位分加成:
  正室参与: 所有属性 +1
  侧室以上参与人数 >= 2: 触发"同心"效果, 额外随机 1 属性

专属组合效果 (特定 NPC 组合):
  "冰玉同心" (小龙女 + 王语嫣): +2 根骨, 战斗 buff 解锁
  "智勇双全" (赵敏 + 黄蓉): +2 悟性, +2 力量
  "三才阵" (3人): +3 体质, 解锁"后宫战法"
  "四象阵" (4人): +4 体质, +1 全属性
  "五行圆满" (5人): +5 全属性, 解锁"万花丛中"被动

风险计算:
  基础风险 = 参与人数 × 10
  和谐度修正 = (100 - 平均和谐度) / 10
  总风险 = 基础风险 + 和谐度修正
  风险触发时: 嫉妒 +20~40, 和谐度 -10~25, 可能触发"嫉妒之夜"事件
```

# 直播/短视频 NSFW 模块详细方案

## 一、概述

### 1.1 模块定位

直播/短视频经济系统是现代纪元 NSFW 模块的核心 P0 模块，承载以下核心功能：
- 主播职业发展路径
- 粉丝-主播关系演化
- MCN 公会运营机制
- 平台规则与流量机制
- NSFW 内容深化与边界

### 1.2 核心特色

**与传统游戏的差异化：**
- 直播间是"半公开私密空间"——表面公开，实际只有粉丝能看到
- 主播与粉丝的关系是多层级的，从"潜水"到"榜一"有明显的权力梯度
- PK 惩罚是 NSFW 内容的天然入口——"输了要接受惩罚"
- 粉丝打赏具有"购买亲密感"的隐喻

### 4.3 PK 惩罚系统

**PK 流程：**
1. 主播 A 向主播 B 发起 PK 邀请
2. 双方开播，计时 5-10 分钟
3. 双方粉丝投票（刷礼物=票数）
4. 结束时票数低者接受惩罚

**惩罚池（按尺度分类）：**

软色情尺度：

- 说暧昧台词

- 湿身诱惑（洒水）
- 局部特写

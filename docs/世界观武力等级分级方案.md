# 世界观武力等级分级修改方案

## 一、现状分析

### 当前世界配置 (`WorldGenConfig`)

```typescript
interface WorldGenConfig {
    worldName: string;
    worldSize: '弹丸之地' | '九州宏大' | '无尽位面';
    dynastySetting: string;
    sectDensity: '稀少' | '适中' | '林立';
    tianjiaoSetting: string;        // 天骄设定（自由文本）
    worldExtraRequirement: string;
    manualWorldPrompt: string;
    manualRealmPrompt: string;
    difficulty: 游戏难度;          // relaxed | easy | normal | hard | extreme
}
```

### 存在的问题

| 问题 | 说明 |
|------|------|
| **缺乏武力等级** | `tianjiaoSetting` 是自由文本，无法标准化控制世界观强度 |
| **低武/中武/高武未区分** | 无法在开局时明确设定世界的武力上限 |
| **修仙未支持** | 当前仅针对武侠，无法选择修仙或玄幻 |
| **提示词生成依赖额外要求** | 需要用户手动在 `worldExtraRequirement` 填写，门槛高 |

## 五、境界体系联动

当 `武力等级 = '修仙'` 时，应自动触发以下调整：

### 5.1 境界体系提示词

自动注入修仙相关内容到 `core_realm`：
```
【修仙补充】
- 灵气：修行根基，天地间弥漫的能量
- 真元：灵气凝练后的修行者内力
- 筑基/金丹/元婴等：修仙境界层次
- 灵根：修行资质，先天禀赋
```

修仙模式下自动包含：
- 灵气分布图

## 附录：超能力/特异功能线 (后续扩展)

### A.1 设计思路

在武力等级体系之外，增加"能力类型"维度，用于区分：
- 传统武侠/修仙体系 → 内力、真气、灵气
- 现代超能力体系 → 超能力、特异功能、基因改造
- 混合世界 → 允许两者并存

### B.5 NSFW 提示词构建

```typescript
// prompts/runtime/nsfw.ts 扩展
import { 武力等级NSFW风格, 超能力NSFW风格 } from '../../models/system';

export const 构建世界观NSFW提示词 = (
    config: {
        武力等级?: 武力等级;
        能力类型?: 能力类型;
        nsfw配置?: NSFW配置结构;
    }
): string => {
    const { 武力等级, 能力类型, nsfw配置 } = config;
    
    // 1. 检查是否启用 NSFW
    if (!nsfw配置?.启用 || nsfw配置?.场景类型 === '无') {
        return '';
    }
    
    // 2. 获取基础风格
    let 基础风格 = '';
    if (能力类型 === '超能力线' && nsfw配置?.超能力分类) {
        基础风格 = 超能力NSFW风格[nsfw配置.超能力分类] || '';
    } else if (武力等级) {
        基础风格 = 武力等级NSFW风格[武力等级] || 武力等级NSFW风格['中武'];
    }
    
    // 3. 根据场景类型添加约束

    switch (nsfw配置.场景类型) {
        case '点到为止':
            场景约束 = `
【场景约束】
- 只描写氛围和情感递进
- 身体接触限于拥抱、牵手、眼神交流

- 场景未自然进入时不应强行触发 NSFW 内容
    `.trim();
};
```
